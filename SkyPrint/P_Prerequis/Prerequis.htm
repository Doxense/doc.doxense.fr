Prérequis

Prérequis techniques

Avant d'entamer la procédure, vérifiez les prérequis suivants :

  • Watchdoc est installé (idéalement sur un autre serveur que le serveur hébergeant SkyPrint). Si Watchdoc SkyPrint autorise l'impression à des visiteurs, Watchdoc doit être configuré pour utiliser la base Invités (Guests) :

     

  • le service MS Windows® Line Printer Daemon (LPD) est installé sur le serveur d’impression Watchdoc (si Watchdoc et Watchdoc SkyPrint sont installés sur des serveurs différents) ;

  • la Console de Supervision de Watchdoc (WSC) est installée et doit obligatoirement :

  • communiquer en TLS :

 

  • son option REST API activée et configurée :

     

  • Les ports suivants sont disponibles :

    • 8080

    • 8443

    • 4712 et 4713

    • 8009

  • une file d'impression (ou plusieurs files) MS Windows® est déclarée sur le serveur hébergeant SkyPrint afin de servir de file de conversion. Cette file (ou ces files) n'est pas contrôlée par Watchdoc. De plus, elle est distincte de la file virtuelle contrôlée par Watchdoc.

Prérequis organisationnels

Avant d'entamer la procédure d'installation et de configuration, vérifiez que vous disposez :

  • de la licence (ou des licences) Watchdoc®, Watchdoc SkyPrint® intégrée à la licence Watchdoc Supervision Console(WSC)® (fournir l'adresse MAC du (ou des) serveur qui héberge SkyPrint) ;

  • des informations de configuration du serveur SMTP :

    • adresse du serveur

    • port

    • protocole

    • courriel générique (disposer d'une adresse mail spécifique à l'envoi des notifications SkyPrint, skyprint@mon_entreprise.com, par exemple) ;

  • des informations de configuration de la boîte de réception des courriels (si Watchdoc SkyPrint autorise l'envoi par mail) :

    • adresse courriel dédiée à Watchdoc SkyPrint® créée par l'administrateur permettant la soumission des documents (prévoir 1 adresse mail par site, i.e. par boîte courriel déclarée dans SkyPrint) ;

    • serveur

    • port

    • protocole

    • compte (utilisateur et mot de passe) de la boîte courriel.

  • des informations de configuration de Watchdoc Supervision Console :

    • serveur

    • port

    • compte REST API

    • serveur Line Printer Remote Protocole (LPR)

    • file LPR par défaut.

  • vous disposez d'une connexion internet afin de télécharger les fichiers d'installation depuis le portail Connect, l'extranet client dédié aux partenaires :

    • setup-skyprint-app-server.exe : ce programme contient l'application serveur de base (JavaEE server wildfly) en tant que serveur ;

    • setup-watchdoc-skyprint.exe : ce programme contient l'application d'impression mobile (Application Java .ear).
