Ce fichier texte, initialement dérivé de JavaScript, est devenu indépendant du langage et de nombreux langages de programmation l’utilisent maintenant. (Source : datasunrise)Db utilisée par Watchdoc pour stocker certaines informations à froid (comme l’avancée d’un scan par exemple). Jusqu’ici, la JsonDB se matérialisait par des fichiers sur disque. Watchdoc/WSC 6.1.0.5138 et WPC for Windows 7.0.4681 - 27/01/2025

Améliorations

  • Watchdoc : minification La minification (ou minimisation) est le processus qui consiste à supprimer tous les caractères inutiles du code source des langages de programmation interprétés ou des langages de balisage sans en modifier la fonctionnalité. Ces caractères inutiles comprennent généralement les caractères d'espacement, les caractères de retour à la ligne, les commentaires et parfois les délimiteurs de blocs, qui sont utilisés pour améliorer la lisibilité du code mais ne sont pas nécessaires à son exécution. La minimisation réduit la taille du code source, ce qui rend sa transmission sur un réseau (par exemple Internet) plus efficace. (Source : Wikipedia) du javascript du portail d’administration de Watchdoc afin d'en renforcer la sécurité
