Agent Watchdoc
(compatible jusqu'à Watchdoc 5.4.1)

Ancien utilitaire permettant de gérer les périphériques d'impression locaux connectés en USB ou adressés directement via IP. (Incompatible avec les versions 5.5 et 6.0)
Fonctionne à condition que le langage d’impression de ces périphériques soit PCL ou PostScrip.

Cf. Manuel d'installation et de configuration de l'Agent Watchdoc.

TeamViewer Outil permettant l'accès à distance pour faciliter la configuration ou les dépannages par le Support Doxense Team Viewer for Doxense

Additional modules for a fee

ScanCare

(cf. versions téléchargeables)

ScanCare® est un module complémentaire qui permet à Watchdoc de gérer la numérisation des documents sur les copieurs et les multifonctions avec WES. Complémentaire à WEScan (la solution de numérisation standard et embarquée de Watchdoc), ScanCare® propose des fonctions permettant d'améliorer le document numérisé et de l'enrichir afin de l'intégrer dans une véritable chaîne documentaire.
