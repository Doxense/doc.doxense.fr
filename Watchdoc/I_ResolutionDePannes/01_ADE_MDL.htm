WES - Scan2USB impossible

2017 - Watchdoc v5.0

Contexte

Depuis certains WES, la fonction Scan2USB s'avère impossible alors qu'elle a bien été activée lors du paramétrage.

Après authentification, l'utilisateur accède à l'écran du périphérique d'impression (illustré ci-dessous sur un WES Toshiba), sur lequel le bouton USB est grisé :

Cause(s)

Ce problème est dû au fait que l'installation du WES a été réalisée avant la mise à disposition des fonctions "Scan2..." dans Watchdoc.

Résolution

Pour résoudre le problème, procédez comme suit :

  1. dans le fichier de configuration Config.xml (enregistré par défaut dans C:\Program Files\Doxense\Watchdoc\Data), remplacez le bloc <template id="PT_QUEUE_USAGE"> existant par le bloc figurant ci-dessous ;

  2. redémarrez le serveur ;

  3. dans la section Profils de sécurité et Droits de Watchdoc, configurez les droits d'accès conformément aux besoins de l'organisation ;

  4. videz le cache des annuaires

  5.
