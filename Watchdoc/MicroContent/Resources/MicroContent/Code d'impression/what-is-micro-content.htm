Code d'impression

Le code d'impression est un code qui permet à l'utilisateur de s'authentifier sur certains WES, en complément de son nom de compte (login). 

 

Il s'agit d'un code alpha-numérique qui doit contenir entre 4 et 16 caractères et ne doit pas contenir plus de 2 chiffres.

 

Les WES compatibles sont : Epson, Hewlett Packard, Konica Minolta, Sharp, Toshiba, Xerox.

 

Ce mode d'authentification doit être configuré :

  • dans l'annuaire LDAP (cf. Configurer l'annuaire LDAP) ;

  • dans l'annuaire META si l'annuaire LDAP y est intégré ;

  • dans le profil WES (cf. Configurer le profil WES du WES concerné).

 

Une fois activé dans l'annuaire et dans le profil WES ce code est saisi par l'utilisateur dans sa page "Mon compte > Codes et badges" (cf. Consulter la page "Mon compte").
