Périphériques d'impression USB

Il arrive que des postes de travail utilisent des périphériques d'impression qui ne sont pas en réseau mais connectés via un port USB (ou port LPT1). Par exemple, lorsque des salariés impriment des documents alors qu'ils sont en télétravail. 
Dans ce cas, il peut être utile que Watchdoc collecte les données d'impression à des fins d'analyse statistiques (ou de comptabilisation). 

 

Avant la version Watchdoc 5.4.1, il était nécessaire d'installer un Agent Watchdoc (ou Agent local) sur le poste de travail pour collecter et envoyer à Watchdoc les données relatives à l'impression.

Au-delà de la version 5.4.1, c'est Watchdoc Print Client for Windows qui, installé sur le poste de travail, est chargé de cette fonction.

 

Notez que les imprimantes connectées en USB ne peuvent pas disposer d'un WES (Watchdoc Embedded Solution) et qu'en conséquence, les fonctions avancées telles que l'impression à la demande, l'authentification sur le périphérique ou WEScan ne sont pas disponibles pour ce type de connexion.

 

Généralement, l'administrateur classe ces files associées aux périphériques USB dans la catégories des "files personnelles" (cf. Configurer une file d'impression).
