Proxy inverse

Les paramètres relatifs au proxy inverse (reverse proxy) sont précisés dans l'interface de configuration Serveur WEB, section DSP.

 

Dans le cas où ce paramètre n'est pas précisé, certaines requêtes peuvent être mal interprétées par Watchdoc. Par exemple, WPC peut ne pas retrouver les périphériques attachés à un emplacement (cf. WPC for Windows - Erreur de détection d'un emplacement).
