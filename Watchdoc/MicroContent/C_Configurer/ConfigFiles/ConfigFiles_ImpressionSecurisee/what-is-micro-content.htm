Impression sécurisée et impression à la demande

Impression sécurisée

L'impression sécurisée est un mode d'impression qui oblige l'utilisateur à débloquer en personne ses demandes d'impressions sur les périphériques après s'y 'être authentifié. 

Ce mode repose sur la fonction "validation" de Watchdoc.

 

Watchdoc permet le mode d'impression sécurisée en bloquant tous les documents dans une file d’impression partagée d'un périphérique. L'utilisateur doit se rendre devant le périphérique, s'authentifier et débloquer ses impression à l'aide du WES.

.

La file d'impression partagée (physique) est configurée sur le poste de l'utilisateur. Plusieurs files peuvent être installées en fonction des besoins d'impression. Dans ce cas, l'utilisateur peut choisir, parmi les files installées sur son poste, celle sur laquelle il souhaite imprimer puis va débloquer ses impressions.

 

Dès qu'il choisit sa file d'impression, l'utilisateur sait sur quel périphérique il doit débloquer son impression.

Impression à la demande

L'impression à la demande permet à l'utilisateur de lancer une demande d'impression sans se soucier de savoir par quel périphérique cette impression est traitée. Il peut ensuite débloquer son impression sur l'un des périphériques compatibles avec la file d'impression. 

 

Pour permettre ce mode, Watchdoc collecte les demandes d'impression dans des files logiques puis dirige les travaux de la file physique sur laquelle l'utilisateur débloque ses impressions. 

 

La file logique est installée sur l'ordinateur de l'usager.

 

L'impression à la demande ne peut être mise en œuvre que si les pilotes de la file Impression à la demande et ceux des périphériques sélectionnés pour le déblocage sont compatibles.

Par ailleurs, cette fonction nécessite de disposer d'une (ou de plusieurs) files supplémentaires dans votre licence Watchdoc (licences FIR).
