File virtuelle

Une file virtuelle est une file qui n'est pas attribuée à un périphérique, mais qui a pour tâche de conserver le travail d'impression prêt à être envoyé au périphérique réseau lorsque l'utilisateur en fait la demande.

 

C'est ce type de file qui est sollicité losque la fonction Impression à la demande Fonction qui permet à l'utilisateur de lancer ses travaux d'impression depuis son environnement de travail puis de débloquer ses impressions sur le périphérique de son choix. La validation de ses impressions peut être effectuée soit depuis l'interface web de Watchdoc, soit depuis le WES, après authentification par l'utilisateur. est activée, ce qui permet à un utilisateur d'envoyer ses travaux d'impression sans désigner d'imprimante spécifique, puis de récupérer ses travaux d'impression sur l'imprimante de son choix une fois qu'il s'est authentifié auprès de celle-ci.

 

Pour configurer une file virtuelle, sélectionnez le groupe ou le mode File virtuelle lors de la configuration de la file d'attente (cf. Configurer une file).

 

Étant donné que la file virtuelle envoie des travaux d'impression à divers périphériques aux capacités très différentes, nous vous recommandons de configurer cette file d'attente avec des capacités maximales (voir la section Périphériques : impression couleur, impression recto verso et grand format) afin d'optimiser son efficacité.
