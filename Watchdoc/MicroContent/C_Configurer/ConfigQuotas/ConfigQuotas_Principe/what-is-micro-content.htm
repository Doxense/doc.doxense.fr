Quotas

Qu'est-ce qu'un quota ?

Dans Watchdoc,

  • un quota est un crédit d'unités de valeurs dédié à l'impression attribué à un utilisateur ou à un groupe d'utilisateurs ;

  • les quotas reposent sur une base de données spécifique, nommée Watchdocpmv nom attribué par défaut à la base de données (type sql) dédiée à la collecte des informations de quotas (crédit attribué à un utilisateur pour l'impression). par défaut (de type SQL®), déclarée et activée lors de l'installation initiale de Watchdoc. La fonction peut être activée et configurée après l'installation initiale.

  • un quota peut être associé à une unité monétaire lorsque les travaux d'impression sont facturables à l'utilisateur. Le quota est alors appelé "PMV : porte-monnaie virtuel". Ce coût monétaire est défini par l'administrateur qui attribue un prix par page en fonction de divers critères (monochrome/couleur, format des feuilles ; recto/verso et applications logicielles dont sont issus les documents) ;

  • un quota s'appuie sur un tarif de refacturation. Il est donc nécessaire d'avoir configuré au moins 1 tarif avant de configurer un quota ;

  • le fonctionnement du quota est défini à l'aide d'une règle de gestion (cf. Configurer les règles de gestion) appliquée sur une file d'impression ;

  • un quota peut être appliqué à un utilisateur ou à un groupe d'utilisateurs (quota de groupes) ;

  • il est possible d'appliquer deux quotas pour une file, de sorte que si le premier quota est épuisé, un deuxième puisse être utilisé ;

  • un quota peut être crédité :

    • par l'administrateur (l'entreprise, l'université, l'administration) manuellement ou automatiquement, selon une fréquence prédéfinie, à l'aide d'une valeur fixe ;

    • par l'utilisateur lui-même, manuellement, selon son besoin.

  • le double-quota permet d'attribuer à un utilisateur un crédit géré par son établissement et un crédit géré par lui-même depuis une station de rechargement ou un site (type Paybox®), par exemple. Dans ce cas, les travaux d'impression sont automatiquement décomptés du deuxième quota lorsque le premier quota est épuisé.

Configurer un quota

Pour configurer un quota :

  1. vérifiez que la fonction est activée et que la base dédiée aux quotas est configurée. Si ce n'est pas le cas, activez la fonction et et configurez la base de données (cf. Activer la fonction Quotas et configurer la base) ;

  2. de créer une file d'impression (ou un groupe de files si le quota s'applique à plusieurs files) ;

  3. de paramétrer un tarif ;

  4. de créer une règle de gestion ;

  5. de créer un quota ;

  6. d'appliquer le quota sur la file d'impression (tarification > paiement automatique)

Où et comment sont utilisés les quotas ?

Les quotas interviennent lors du déblocage des travaux d'impression, soit depuib:

  • soit depuis le WES ;

  • soit depuis la page "Mon compte".

Avant de débloquer ses impressions, l'utilisateur peut ainsi vérifier si son solde est suffisant pour imprimer.

Recharger un quota

Il est possible de configurer la manière dont est rechargé un quota (cf. Configurer la base des Quotas).

Les quotas peuvent être rechargés manuellement par l'administrateur ou automatiquement.

La fréquence de réinitialisation (rechargement) est configurable dans la règle de gestion (cf. Configurer une règle de gestion). 

Si l'administrateur l'autorise à recharger son compte, l'utilisateur peut le faire depuis l'interface web Mon Compte ou depuis une station de rechargement ou un site web dédié (type PayBox®, par exemple).
