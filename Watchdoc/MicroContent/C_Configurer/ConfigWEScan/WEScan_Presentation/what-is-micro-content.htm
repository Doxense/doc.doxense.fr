WEScan

Watchdoc WEScan est une application qui permet à l'utilisateur de numériser des documents depuis un périphérique.

 

Une fois le scan réalisé, il est envoyé vers le serveur Watchdoc qui se charge de l'envoyer vers la destination définie dans le profil de destination (adresse mail ou dossier de l'environnement de travail de l'utilisateur). Cf. WEScan Présentation pour plus d'informations).

 

Les marques compatibles avec WEScan sont (cf. WEScan Prérequis pour les détails) :

- Hewlett Packard

- Konica Minolta OpenAPI

- Kyocera

- Lexmark SDK6

- Sharp (uniquement les modèles compatibles avec le WES)

- Xerox
