Watchdoc - Mettre à jour la clé de licence

Contexte

Si vous mettez à jour Watchdoc / WSC à partir d'une version antérieure à la 6.1.0 ou lorsque la configuration de votre parc d'impression change (ajout de serveurs, de périphériques, de WES, notamment), Doxense® vous fournit une nouvelle licence associée à la nouvelle version ou aux nouveaux modules livrés (WES, FIL, FIR, etc.) répondant à vos besoins.

 

Cette nouvelle licence doit être enregistrée en lieu et place de la précédente.

Procédure

Prérequis

Enregistrez le fichier de licence (..wlk) fourni par Doxense sur le serveur hébergeant Watchdoc ou dans un espace de travail librement accessible depuis ce serveur. 

Mettre à jour la licence automatiquement

Pour mettre à jour la licence Watchdoc :

  1. accédez à l'interface d'administration de Watchdoc en tant qu'administrateur ;

  2. depuis le Menu Principal, section Configuration, cliquez sur Configuration Avancée ;

  3. dans l'interface Configuration avancée, cliquez sur Licence serveur :

 

èVous accédez à l'interface Licence serveur dans laquelle sont détaillées les licences en cours et la configuration réseau.

  1. dans la section Modes d'activation, paramètre Fichier de licence, cliquez sur le bouton Parcourir ;

  2. parcourez votre espace de travail et sélectionnez le fichier de licence fourni par Doxense (extension .wlk) :

 

  1. une fois le fichier sélectionné, cliquez sur le bouton Envoyer le certificat de licence..

 

èLorsque l'opération se déroule correctement, Watchdoc vous renvoie à l'interface d'administration principale.

èLorsque l'opération échoue, un message vous informant qu'un problème est sur- venu s'affiche dans l'interface. Dans ce cas, vous pouvez tenter de procéder à la mise à jour manuelle de la licence.

Mettre à jour la licence manuellement

Si l'opération a échoué, vous pouvez procéder à la mise à jour manuelle de la licence :

  1. ouvrez le fichier de licence (extension .wlk) fourni par Doxense à l'aide d'un éditeur de texte (type notepad.exe) ;

  2. dans l'encart Saisie manuelle, copiez le bloc de texte compris entre les mentions # BEGIN LICENSE et # END LICENSE incluses ;

  3. cliquez sur le bouton Enregistrer le certificat de licence pour valider la saisie de la licence :

 

èLorsque  l'opération  se  déroule  correctement,  Watchdoc vous renvoie à l'interface d'administration principale.

èLorsque l'opération échoue, un message vous informant qu'un problème est survenu s'affiche dans l'interface. Dans ce cas, contactez le Support Doxense via Connect.
