Watchdoc - Restaurer une version antérieure de Watchdoc

Principe

Après mise à jour de Watchdoc, il peut arriver que l'on souhaite revenir à la version antérieure. Il est alors nécessaire de restaurer la version précédemment installée. 

 

Cette opération est possible car, lorsqu'on lance une mise à jour, l'assistant génère automatiquement des fichiers de sauvegarde de la version précédemment installée. Par défaut, les fichiers de sauvegarde sont générés et enregistrés dans le dossier contenant les fichiers d'installation de la nouvelle version :



A partir de la version 6.1.0.5122, vous pouvez aussi lancer la mise à jour en ligne de commande en précisant le dossier dans lequel vous souhaitez enregistrer les fichiers de sauvegarde (cf. Mettre à jour en ligne de commande).

 

La restauration consiste à copier les fichiers de l'ancienne version pour les coller dans les dossiers de la mise à jour afin de les remplacer.

Prérequis

Pour restaurer une version antérieure de Watchdoc, il est impératif d'avoir conservé les deux fichiers de sauvegarde backup_kernel_[n°version_date].zip et backup_web_[n°version_date].zip automatiquement générés par l'assistant lors de la mise à jour et d'y avoir accès.

Procédure

  1. En tant qu'administrateur, accédez au serveur sur lequel est hébergé Watchdoc ;

  2. accédez au dossier dans lequel sont enregistrés les 2 fichiers de sauvegarde ;

  3. décompressez l'archive backup_kernel_[n°version_date].zip ;

  4. décompressez ensuite l'archive backup_web_[n°version_date].zip ;

  5. depuis la console des services (ou depuis l'invite de commande), stoppez les services suivants :

    • Watchdoc.exe

    • WatchdocPrivilegedService.exe

    • Windows Process Activation Service (WAS)

    • World Wide Web Publishing Service (W3SVC) :


     

  6. rendez-vous dans le dossier décompressé backup_kernel_[n°version_date] et copiez tous les fichiers qui y figurent ;

  7. collez tous les fichiers dans le dossier Watchdoc (installé par défaut dans C:\Program Files\Doxense\Watchdoc) pour remplacer tous les fichiers de la nouvelle version par ceux de l'ancienne :



  8. rendez-vous dans le dossier décompressé backup_web_[n°version_date] et copiez tous les fichiers qui y figurent ;

  9. collez tous les fichiers dans le dossier web de Watchdoc (par défaut C:\inetpub\wwwroot\watchdoc) pour remplacer tous les fichiers de la nouvelle version par ceux de l'ancienne :



  10. lorsque tous les fichiers ont été remplacés, retournez dans la console (ou avec une invite de commande) pour relancer les services suivants :

    • Watchdoc.exe

    • WatchdocPrivilegedService.exe

    • Windows Process Activation Service (WAS)

    • World Wide Web Publishing Service (W3SVC) :

  11. Accédez à l'interface d'administration Watchdoc pour vérifier que le numéro de version est bien celui de l'ancienne qui vient d'être restaurée.
