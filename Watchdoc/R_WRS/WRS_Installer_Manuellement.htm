WRS - Installer manuellement Watchdoc for Reporting Services

Dans le cas où vous installez le jeu de rapports WRS® pour un serveur SQL 2012 ou supérieur, la procédure d'installation est la suivante :

  1. créer un dossier;

  2. créer une source de données ;

  3. déployer des rapports.

Créer le dossier

  1. Accédez au Gestionnaire de rapports de: Reporting Services (http://nomduserveur/Reports_WATCHDOC)

  2. Cliquez sur Nouveau dossier ;

     

  3. Dans la boîte Nouveau dossier, indiquez le nom du dossier dans lequel seront enregistrés les rapports WRS :

     

  4. Cliquez sur le bouton OK pour valider l'ajout du dossier.

  5. Cliquez sur le dossier créé : dans le formulaire, complétez les champs suivants :

  6. Nom : indiquez dans ce champ le nom de la base de données ;

  7. Chaîne de connexion :indiquez dans ce champ :

  8. Data Source= indiquez le chemin menant à l'instance de la base de données ;

    Initial Catalog= indiquez le nom de la base de données.

  9. Sécurité intégrée de Windows :

  10. cochez le bouton radio correspondant au mode d'authentification adopté ;

  11. Clquez sur le bouton Tester la connexion pour vérifier la validité des informations fournies.

  12. cliquez sur le bouton Appliquer.


 

Déployer les rapports

  1. Dans SQL Server Reporting Services, rendez-vous dans le dossier Watchdoc ;

  2. Dans le menu du dossier, cliquez sur le bouton Télécharger un fichier ;


     

  3. Cliquez sur Browse et parcourez votre espace de travail pour sélectionnez le premier rapport .rdl (nommé Reports dans le sous-dossier WRS ;

  4. cliquez sur OK pour valider le choix du rapport :

     

  5. Réitérez l'opération pour chacun des fichiers .rdl présent dans le dossier Reports.

  6. Une fois terminé, vous pouvez changer la vue en cliquant sur Mode détail en haut à droite pour afficher les rapports sous forme de liste.

     

  7. Cliquez sur le bilan Statement pour ouvrir le gestionnaire de rapports ;

  8. dans le gestionnaire de rapports, précisez la période pour laquelle vous souhaitez connaître les statistiques, puis cliquez sur Afficher le rapport.



→ La mise en place du rapport WRS est terminée.
