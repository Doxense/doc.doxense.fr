Gérer les files d'impression

Consulter les informations d'une file d'impression

Pour accéder aux informations relatives à l'usage d'une file d'impression :

  1. depuis l'interface d'exploitation des files, dans la liste Files d'impression, cliquez sur la file dont vous souhaitez consulter les données ;

  2. vous accédez à l'interface de gestion de la file ;

  3. dans l'onglet Etat, consultez les informations relatives aux compteurs machine, aux alertes en cours et aux niveaux de consommables :


Modifier les propriétés d'une file d'impression

Pour modifier les propriétés d'une file d'impression :

  1. depuis l'interface de gestion de la file d'impression, cliquez sur le bouton Editer les propriétés ;

 

  1. dans le formulaire Propriétés de la file d'impression, modifiez les données souhaitées ;

  2. cliquez sur le bouton pour valider les modifications.

Suspendre une file d'impression

Pour suspendre temporairement une file d'impression (utile, par exemple, dans le cas où un périophérique doit être dépanné) :

  1. depuis l'interface de gestion de la file d'impression, cliquez sur le bouton pour la suspendre ;

  1. lorsque la file est suspendue, le bouton est grisé et le bouton s'affiche.

Surpris par l'indisponibilité de la file lors de sa suspension, certains utilisateurs ont peut-être multiplié les demandes d'impressions. Il est donc recommandé de vérifier le contenu de la file d'impression avant de la redémarrer, afin d'éviter une surcharge des imprimantes.

Redémarrer une file suspendue

Après suspension d'une file, vous pouvez la redémarrer :

  1. dans l'interface de gestion de la file, lorsqu'elle est désactivée, le bouton est grisé et n'est plus activable ;

  2. cliquez sur le bouton pour la redémarrer ;

     

  1. lorsque la file est redémarrée, le bouton est de nouveau activé.

Supprimer une file d'impression

  1. Dans l'interface de gestion de la file, cliquez sur le bouton Supprimer la file ;

 

  1. Dans la boîte de confirmation qui s'affiche, précisez si vous souhaitez conserver une archive de la configuration ou la supprimer définitivement ;

  2. cliquez sur le bouton Supprimer pour valider votre choix.

Il peut être utile d'archiver la configuration d'une file désactivée, de sorte qu'elle puisse être réutilisée ultérieurement.
