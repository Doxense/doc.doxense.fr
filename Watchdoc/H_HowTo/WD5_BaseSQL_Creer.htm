Base de données SQL pour Watchdoc - Créer

Procédure

Pour créer la base de données :

  1. lancez l'outil Microsoft® SQL Server Management Studio ;


  2. depuis SQL Server Management, connectez-vous au serveur SQLde Watchdoc ;

  3. sélectionnez l'entrée Bases de données, puis cliquez droit pour accéder au menu ;

  4. sélectionnez Nouvelle base de données...

  1. dans l'interface Nouvelle Base de données, saisissez le Nom de la base de données dans la zone dédiée ;

  2. puis cliquez sur le menu Options et choisissez le Mode de récupération Simple :


  1. cliquez sur OK pour valider la création de la base de données.

è Une fois la base de données créée, vous pouvez la configurer dans Watchdoc.
