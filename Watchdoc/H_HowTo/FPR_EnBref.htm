Watchdoc / WSC - Actions rapides

Vérifier si les WES traces (logs) sont activés

Objectif : permet de vérifier si les WES traces sont activées et de les télécharger, de les activer ou de les désactiver au besoin.

Procédure

  1. Depuis le Menu principal, section Exploitation, cliquez sur Files d'impression...

  2. Sélectionnez la file concernée.

  3. Dans l'interface de gestion de la file, cliquez sur l'onglet Propriétés

Vérification : dans la section WES, la valeur du paramètre WES traces indique le statut des logs : activés ou désactivés. Vous pouvez alors activer, désactiver ou télécharger les logs à l'aide des boutons figurant dans cette section.

 

Déterminer la version du .NET Framework d'un ordinateur / serveur

Objectif : permet de déterminer la version du .NET Framework installé sur votre ordinateur/serveur.

Procédure :

  1. depuis la zone de recherche de la barre des tâches, entrez "regedit"

  2. dans l'Editeur du Registre, déployez l'arborescence suivante : Ordinateur\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\NET Framework Setup\NDP\v4\Full

  3. vérifiez la version affichée dans la colonne Données.

 

Activer la Télémétrie dans Watchdoc

Objectif : permet de soumettre Watchdoc à la Console de Supervision WSC

Procédure

  1. Depuis le Menu principal, section Configuration, cliquez sur Configuration avancée.

  2. Cliquez sur Configuration Système.

  3. Cochez la case de la section Télémétrie (ou case Activer, en fonction de la version).

  • Dans le champ Hôte, indiquez le nom du serveur qui héberge WSC, ainsi que le numéro de port.

  • Cochez la case HTTPS.

  1. Validez la configuration.

Vérification : dans WSC, cliquez sur le menu Serveurs et vérifiez que le serveur sur lequel est installé Watchdoc est bien supervisé par WSC (marqueur vert ou orange).

 

Trouver le numéro de série d'un périphérique d'impression

Objectif : Retrouver le numéro de série d'un périphérique d'impression, notamment pour compléter le champ Identifiant du WES dans le profil WES.

Procédure

  1. Depuis le Menu principal, section Exploitation, cliquez sur Files d'impression,....

  2. Dans la liste des files, cliquez sur la file dont vous souhaitez connaître le numéro de série.

  3. Sous l'onglet Etat, section Monitoring, repérez le numéro de séire.

    Vous pouvez alors le copier puis le coller dans un autre champ de paramétrage.

 

Modifier le mot de passe de maintenance

Objectif : permet de sécuriser Watchdoc

Procédure :

  1. Depuis le Menu principal, section Configuration, cliquez sur Configuration avancée.

  2. Cliquez sur Configuration Système.

  3. Dans la section Sécurité, paramètre Super Utilisateur, saisissez le nouveau mot de passe.

  4. Cochez la case Interdire l'accès depuis le réseau pour obliger l'administrateur à se connecter à Watchdoc depuis le serveur sur lequel il est installé. 

  5. Cliquez sur Valider pour enregistrer la modification. 

Vérification : relancez Watchoc et tentez de vous y authentifier avec l'ancien mot de passe. Un message vous informe que le mot de passe est incorrect.

 

Exclure une file de WPC for Windows

Objectif : permet qu'une file ne soit pas prise en compte par WPC for Windows

Procédure :

  1. Depuis le Menu principal, section Exploitation, cliquez sur Files d'impression, emplacements, groupes de files & pools.

  2. Dans la liste des files, repérez la file à exclure et cliquez sur le bouton pour éditer ses propriétés. 

  3. Dans la section Plateforme, paramètre Tags supportés, saisissez le tag !WIN32

  4. Cliquez sur Valider pour enregistrer la modification apportée au paramétrage de la file.

Vérification : lancez WPC for Windows et vérifiez que la file exclue ne figure plus dans la liste des files d'impression installées

 

Permettre le déblocage des impressions même en cas de panne de la bdd interserveur

Objectif : lorsque l'impression à la demande interserveur est activée et configurée, permet aux utilisateurs de s'authentifier sur le WES et de débloquer leurs impressions en attente même si la base de données interserveur est en panne.

Procédure :

  1. Depuis le Menu principal, section Configuration, cliquez sur Configuration avancée.

  2. Cliquez sur Configuration système. 

  3. Dans la section Impression interserveur, activez le Cache et indiquez la durée du fusible.

  4. Valider la configuration du serveur. 

Vérification : lancez une impression, arrêtez la base de données interserveur et vérifiez que vous arrivez à vous authentifier sur le WES puis à débloquer l'impression lancée avant l'arrêt de la bdd.

 

Activer l'impression en mode livret sur un périphérique d'impression

Objectif : permet d'imprimer en mode "livret"

Procédure :

  1. Depuis le gestionnaire d'impression MS Windows (print manager), sélectionnez l'imprimante sur laquelle vous souhaitez activer l'option Livret.

  2. Cliquez sur Propriétés.

  3. Cliquez sur l'onglet Avancées. 

  4. Cochez la case Activer les fonctionnalités d'impression avancées. 

  5. Cliquez sur OK pour valider le paramétrage.

Vérification : lancez l'impression d'un document d'au-moins 4 pages. Dans les options d'impression > onglet Finition, vérifiez que vous disposez de l'option Mise en page livret.

 

Masquer les codes utilisateur aux opérateurs qui gèrent l'annuaire

Objectif : permet de masquer les codes d'identication (PUK, PIN, etc.) aux opérateurs en charge de gérer les comptes utilisateurs dans l'annuaire. Les codes sont alors remplacés par des étoiles.

Depuis la version 6.1.4926, les codes d'identification (PUK, PIN, etc.) sont inclus dans les données couvertes par le rôle d'administration "Visualisation des identités des utilisateurs".

Procédure :

  1. accédez à l'interface d'administration de Watchdoc en tant qu'administrateur ;

  2. depuis le Menu principal, section Gestion, cliquez sur Droits d'accès ;

  3. dans la liste des rôles d'administration, vérifiez que le rôle Visualisation des identités des utilisateurs ne s'applique pas aux opéateurs ;

  4. déconnectez-vous de l'interface d'administration de Watchdoc ;

  5. reconnectez-vous à l'interface d'administration de Watchdoc en tant qu'opérateur ;

  6. depuis le Menu principal, section Configuration, cliquez sur Annuaires utilisateurs ;
