Watchdoc - Spools - Archiver des spools - Procédure

Principe

L'archivage des spools d'une file d'impression nécessite plusieurs étapes de configuration :

  1. activation de l'archivage des spools au niveau de Watchdoc ;

  2. activation de l'archivage des spools sur la file spécifique à analyser ;

  3. application d'une règle permettant de préciser la nature des spools à archiver et la fréquence d'archivage.

     

Une fois les spools collectés pour analyse, il convient de reconfigurer Watchdoc comme dans son état initial de manière à ne pas encombrer le dossier Archives de spools inutiles.

Procédure

Configurer l'archivage des spools

Pour configurer l'archivage des spools :

  1. accédez à l'interface d'administration Watchdoc en tant qu'administrateur ;

  2. depuis le Menu principal, section Configuration, cliquez sur Configuration avancée ;



  3. Dans l'interface Configuration avancée, cliquez sur Archivage des documents :



  1. Dans l'interface Configuration de l'archivage :

    1. cochez la case Paramètres ;

    2. dans le champ Répertoire, saisissez le chemin du répertoire dans lequel vous souhaitez enregistrer les archives de spools ;

    3. cochez la case Compresser les documents archivés.

  1. Validez la configuration de l'archivage en cliquant sur le bouton .

Paramétrer la file d'impression

Pour paramétrer la file d'impression :

  1. depuis le Menu principal, section Exploitation, cliquez sur Files d'impression, groupes de files & pools ;

  2. dans la liste des files, sélectionnez la file sur laquelle ont été détectés les plus importants écarts (une file par driver) ;

  3. Cliquez sur le bouton Editer les propriétés ;

  4. Dans l'interface Propriétés de la file, section Archivage, configurez l'archivage des spools :

    1. cochez la case Utiliser une configuration spécifique ;

    2. cochez la case Autoriser les archives temporaires ;

    3. dans les menus déroulants, sélectionnez la fréquence d'archivage des spools :

       

  1. cliquez sur le bouton pour enregistrer la nouvelle configuration de la file. 

→ Au terme de ce paramétrage, les travaux d'impression sont stockés dans le répertoire Archives du dossier Watchdoc.

Appliquer une règle

Afin de permettre à Watchdoc de n'archiver que certains travaux d'impression, il convient d'appliquer une règle permettant de désigner les travaux d'impression concernés par l'archivage :

  1. depuis l'interface de gestion de la file, cliquez sur l'onglet Règles ;

  2. cliquez sur le bouton Editer de la file ;

  3. sélectionnez ensuite le filtre prédéfini Toujours pour capturer l'ensemble des documents ;

  4. sélectionnez l'onglet Règles de la file d'impression concernée, puis cliquez sur Editer :



     

  1. Dans la section Soumission d'un document, dans la liste des filtres, sélectionnez le filtre prédéfini Toujours afin de capturer l'intégralité des documents :



  1. cliquez sur le bouton pour valider le paramètrage de la règle sur la file ;

  2. lancez un travail d'impression sur cette file pour vérifier que le spool est correctement enregistré dans le dossier d'archives de Watchdoc (C:\Program Files\Doxense\Watchdoc\Archives par défaut) ;

  3. collectez les fichiers de spools dans le dossier Archives afin de les analyser ou de les envoyer au Support Doxense® pour analyse.
