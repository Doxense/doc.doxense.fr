Watchdoc - File - Créer et gérer une file virtuelle

Principe

La file virtuelle ("virtual queue") est une file facultative déclarée par l'administrateur Watchdoc pour répondre à des besoins autres que ceux assurés par la file d'impression physique. Cette file virtuelle n'est associée à aucun périphérique en particulier, d'où le port NUL qui la caractérise. Comme la file d'impression physique, elle a pour fonction de générer le spool File d'attente avant envoi au périphérique. Le mot 'spool' est, à l'origine, l'acronyme de 'simultaneous processing operations on line'. Le spool d'impression désigne le fichier d'édition envoyé au périphérique d'impression. Il regroupe l'ensemble des ordres que l'imprimante doit exécuter pour mener à bien l'impression d'un ou plusieurs documents. correspondant aux travaux d'impression qui lui sont envoyés.

 

La file virtuelle est créée pour répondre à deux besoins précis :

  1. permettre, pour un périphérique physique, de définir des caractéristiques autres que celles qui ont déjà été définies dans la file d'impression afin d'offrir plusieurs options de finition au périphérique. Concrètement, si vous disposez d'une imprimante réseau dotée de 2 bacs, l'un pour le papier A4 standard et l'autre pour le papier A4 à en-tête, la file virtuelle permet de mettre le 2e bac à disposition de l'utilisateur. Pour ce périphérique unique, vous définissez :

    • une file partagée configurée pour utiliser le bac de papier A4 standard comme bac par défaut ;

    • une file virtuelle configurée pour utiliser le bac de papier A4 avec en-tête.

    • Lorsque l'utilisateur lance son impression, il choisit la file d'impression qui convient à son besoin.

       


  1. permettre l'impression "à la demande", c'est-à-dire sur n'importe quel périphérique (non prédéfini par défaut.). L'utilisateur a donc la liberté de choisir son périphérique d'impression non pas au moment où il lance l'impression de son poste de travail, mais au moment où il valide ses impressions. Les travaux sont donc stockés dans la file virtuelle en attendant d'être envoyés vers un périphérique spécifique :

Prérequis

Avant de configurer les files virtuelles, il convient de vérifier les prérequis suivants :

  • Watchdoc est opérationnel ;

  • les périphériques sont installés et configurés ;

  • les files d'impression sont configurées ;

  • lorsqu'elles servent à l'impression "à la demande", les files virtuelles doivent être dotées d'un pilote compatible avec les périphériques vers lesquels elles sont susceptibles d'être envoyées. 

  • lorsqu'elles servent à l'impression "à la demande", les files d'impression et les files virtuelles doivent être situées sur le même serveur, les spools d'impression ne pouvant pas être transférés d'un serveur à l'autre.

Procédure

Créer une file virtuelle

Pour créer une file virtuelle :

  1. depuis le serveur, accédez au Gestionnaire d'impression (Print Management). Vous pouvez exécuter printmanagement.msc pour y accéder ;

  2. dans l'outil Gestion de l'impression ,cliquez sur serveur d'impression > Ports > Ajouter un port :

     

  1. dans la fenêtre Port d'imprimante, sélectionnez le type de port Local :

     

  1. dans la fenêtre Nom du port, saisissez la valeur NUL ;

  2. validez en cliquant sur OK puis Fermer :

     

  1. cliquez ensuite sur Fermer pour revenir à l'outil Gestion de l'impression :

  2. depuis le menu Serveurs d'impression, cliquez-droit sur Imprimante > Ajouter une imprimante ;

  3. dans la boîte Installation de l'imprimante, sélectionnez Ajouter une nouvelle imprimante via un port existant, puis sélectionnez le port NUL :

  4. cliquez sur Suivant :

     

  1. dans la liste Pilote d'imprimante, sélectionnez un pilote compatible avec les périphériques vers lesquels la file virtuelle peut envoyer les travaux d'impression ;

  2. cliquez sur Suivant:

  3. dans la boîte Nom de l'imprimante, attribuez un nom explicite à la file virtuelle ;

  4. cochez la case Partager cette imprimante, puiscliquez sur Suivant ;



     

  1. La file d'impression virtuelle est ajoutée sur le serveur. Vous pouvez ensuite la configurer dans Watchdoc.

     

Configurer la file virtuelle

Pour configurer la file virtuelle :

  1. depuis le Menu Principal de Watchdoc, section Configuration, cliquez sur Imprimantes et périphériques ;



  2. dans la liste Imprimantes & périphériques, sélectionnez la file virtuelle créée lors de l'étape précédente ;

  3. dans l'interface Propriétés de la file d'impression, section Informations générales complétez les paramètres suivants :

    • Mode : sélectionnez le mode Validation ;

    • Virtuelle : cochez la case File virtuelle pour impression sécurisée et/ou à la demande :

       

  4. dans la section Redirections, sélectionnez, en Cibles, la ou les files d'impression vers lesquelles peuvent être envoyés les travaux d'impression :







  5. dans la section Mode Expert, cochez la case Chiffrer le titre des documents... si vous souhaitez que le titre des travaux d'impression n'apparaisse pas en clair dans la file d'attente partagée :
