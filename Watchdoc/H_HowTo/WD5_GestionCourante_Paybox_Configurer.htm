Watchdoc - Configurer le site de paiement en ligne Paybox Solution de paiement en ligne commercialisée par Verifone e-commerce permettant à des utilisateurs soumis à des quotas de payer leurs impressions en ligne.

Prérequis

Votre licence Watchdoc doit comporter l'option Paybox pour autoriser la configuration du paiement en ligne Paybox.

Si la section Paybox n'apparaît pas dans l'interface de Configuration Système, c'est que cette condition n'est pas remplie. Dans ce cas, contactez votre référent commercial Doxense ou le partenaire revendeur.

Pour configurer le mode de paiement Paybox, il est nécessaire d'avoir activé et configuré la base de quotas.

Procédure

Pour configurer le paiement via Paybox®, il convient :

  1. de paramétrer le Connecteur Paybox ;

  2. de paramétrer le mode de rechargement dans la page Mon Compte (profil de déblocage web) de Watchdoc s'il est autorisé.

Paramétrer le Connecteur Paybox

  1. depuis le Menu Principal de Watchdoc, section Configuration, cliquez sur Configuration avancée ;

  2. dans l'interface Configuration avancée, cliquez sur Configuration système ;

  3. dans la section Paybox, complétez les champs à l'aide des paramètres suivants :

    • Numéro de site : saisissez le numéro de site fourni par Paybox ;

    • Numéro de rang : saisissez le numéro de rang fourni par Paybox ;

    • Identifiant interne : saisissez l'identifiant interne fourni par Paybox ;

    • Clé secrète : saisissez la clé fournie par Paybox et mise à votre disposition dans l'espace client (Back office) ou la clé de test suivante : 0123456789ABCDEF0123456789ABCDEF0123456789ABCDEF0123456789ABCDEF0123456789ABCDEF0123456789ABCDEF0123456789ABCDEF0123456789ABCDEF;

    • URL externe ; saisissez l'adresse IP (ou le FQDN si vous utilisez le port 443 et le certificat) et le port (80, 8080-8085 (HTTP) ou 443 (HTTPS)) via lesquels accéder au site Paybox ;

    • URL Paybox : saisissez l'URL d'appel de la page de paiement Paybox ;

    • Contrôles de sécurité :

      • adresses : cochez la case pour limiter aux seules adresses Paybox la confirmation de paiement. Décochez la case si la confirmation de paiement sollicite un autre système et indiquez l'IP de ce système au besoin (IP d'un reverse proxy, par exemple) ;

      • vérifier la signature : cochez la case pour vérifier la signature avec le certificat public Paybox.

         

Paramétrer la page Mon Compte :

  1. depuis le Menu Principal de Watchdoc, section Configuration, cliquez sur Web, WES & Destinations de numérisation ;

  2. dans la section Profils déblocage web et mobile, sélectionnez la page Web Mon compte concernée par le rechargement via Paybox® et cliquez sur le bouton pour l'éditer ;

  3. dans la section Paiement, sélectionnez le mode de paiement Quota Crédit d'unités de valeurs dédié à l'impression et attribué à un utilisateur ou à un groupe d'utilisateurs. ;

  4. cochez la section Autoriser le rechargement des quotas (si la case ne peut pas être cochée, c'est que la base de quotas n'est pas correctement configurée) ;

  5. En fonction du mode de rechargement choisi (crédit ou débit), sélectionnez le support qui sera débité, soit un PMV, soit une carte. En fonction de ce choix, la liste permettant de forcer le débit sur un compte particulier est mise à jour. Vous pouvez alors sélectionner (ou non) le compte qui sera obligatoirement débité.
    Si vous ne précisez aucun compte, l'utilisateur peut sélectionner lui-même le compte qu'il souhaite débiter.

  1. dans la colonne Montants en cents, saisissez les montant en centimes, en évitant les virgules et les points, Paybox ne supportant pas les montants comportant des virgules et des points.

Recharger le compte à l'aide de la page dédiée Rechargement via Paybox

N.B. : le rechargement du compte n'est disponible qu'aux utilisateurs présents dans l'annuaire (A.D.).

  1. A l'aide d'un lien qui lui est fourni par l'administrateur, l'utilisateur se connecte à la page Watchdoc Rechargement via Paybox ;


 

  1. dans cette interface, il sélectionne le compte à recharger et choisit dans la liste déroulante le Montant du rechargement ;

  2. dans le champ dédiée, il saisit son Adresse e-mail ;


Ces valeurs sont obligatoires : si les champs restent vides, l'utilisateur ne peut pas valider l'opération et le message "Une erreur s'est produite" s'affiche.

 

  1. il clique sur Suivant pour valider les informations saisies ;

  2. Paybox envoie la transaction à Watchdoc via le site web d'administration. Cette transaction est enregistrée dans un fichier de traces pour diagnostic en cas de besoin. Le PMV de l'utilisateur est alors alimenté par le service Watchdoc du montant choisi.
