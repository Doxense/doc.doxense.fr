Calculateur d'économies - Utilisation

Utiliser le calculateur d'économies

  1. Pour procéder à l'estimation des économies réalisables, définissez tout d'abord la période sur laquelle appliquer les critères : dans la section Sélection de la période, choisissez dans les listes déroulantes les mois et années déterminant la période sur laquelle porte l'estimation ;

  1. cliquez sur le bouton Soumettre :

 

Le nombre de pages prises en compte dans le calcul est affiché dans un encart et l'interface de calcul s'affiche :

Utilisation du Calculateur

Dans l'interface sont affichées les données relatives aux impressions effectuées par les utilisateurs au cours de la période sélectionnée et répondant aux paramètres actuels de Watchdoc®.

Pour estimer les économies réalisables sur ce jeu de données :

  • appliquez un tarif spécifique ;

  • modifiez les paramètres d'impression.

Appliquer un tarif

Pour estimer les économies réalisables, il est nécessaire d'appliquer un tarif aux impressions :

  1. dans la section Sélection du tarif, sélectionnez dans la liste déroulante l'un des tarifs configurés dans Watchdoc ;


  2. ou cliquez sur le bouton pour accéder à la liste de tous les tarifs disponibles et cliquez sur l'un des tarifs pour le sélectionner ;

 

Si nécessaire, vous pouvez créer de nouveaux tarifs :

  1. cliquez sur le bouton pour accéder à la liste des tarifs disponibles ;

  2. en bas de la liste, cliquez sur le lien Créer un nouveau tarif simple ;

  3. dans l'interface dédiée, saisissez les données nécessaires à la création du tarif (cf. Fiche pratique - Créer et configurer un tarif), puis cliquez sur le bouton pour en Valider la création ;

  4. sélectionnez ensuite ce tarif nouvellement créé pour constater les économies réalisables sur vos impressions.

  5. Le détail du tarif s'affiche à droite du tarif sélectionné

  6. cliquez sur le bouton Soumettre et consultez la section Economies pour constater les économies réalisées :

Modifier les options et transformations de spools

Pour estimer les économies réalisables, modifiez les paramètres d'impression à l'aide du curseur.
Pour tous ces paramètres, le curseur est initialement placé sur le jalon Actuel qui désigne l'activité d'impression telle qu'elle est paramétrée dans Watchdoc :

  1. paramètres de transformation de spools (Couleur vers noir et blanc : et Recto simple vers recto-verso ):

    • Suggérée : permet de constater les économies réalisées si la moitié des utilisateurs accepte la modification de leurs travaux :

    • Obligatoire : permet de constater les économies réalisées si la totalité des utilisateurs accepte la modification de leurs travaux.

  1. autres paramètres (Photocopie vers Scan et Fax vers Scan :

    • Faible : permet de constater les économies réalisées si 25 % des utilisateurs acceptent la modification de leurs travaux ;

    • Moyen : permet de constater les économies réalisées si 50 % des utilisateurs acceptent la modification de leurs travaux ;

    • Elevé : permet de constater les économies réalisées si 50 % des utilisateurs acceptent la modification de leurs travaux.

  1. Consultez ensuite l'encart dans lequel s'affichent les Economies réalisables :

  1. Pour affiner l'analyse, cliquez sur le bouton Calculateur d'économies avancé pour accéder à un outil permettant de modifier plus précisément les paramètres d'impression.
