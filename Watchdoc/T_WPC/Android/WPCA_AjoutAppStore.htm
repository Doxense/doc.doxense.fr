Watchdoc Print Client for Android (app. mobile) - Ajouter l'application dans le Store Android

Principe

Lorsque Doxense reçoit votre ID d'organisation, il est ajouté à la liste des organisations / entreprises autorisées à utiliser Watchdoc. Dès lors, vous pouvez donc retrouver Watchdoc dans les applications disponibles pour votre entreprise et l'ajouter.

Procédure

Ajouter l'application Watchdoc

Pour permettre à vos utilisateurs de télécharger l'application depuis votre store Android, vous devez l'ajouter dans les applications de votre entreprise.

  1. Depuis l'interface Google Workspace Admin, dans le menu de gauche, cliquez sur Applications > Applications Web et mobiles.

  2. Dans l'interface Applications, cliquez sur Ajouter une application > Rechercher des applications.

  3. Dans l'interface Rechercher dans les applications, saisissez la valeur Watchdoc.

  4. Lorsque l'application Watchdoc s'affiche dans la liste des résultats, cliquez sur Installer.

  5. Une fois installée, l'application s'affiche dans la liste des applications disponibles : vos utilisateurs pourront l'installer sur leur mobile dès qu'ils accéderont à Google Play store avec leur compte professionnel.

Configurer l'url du serveur de découverte dans l'application

Pour permettre à l'application Watchdoc d'envoyer les impressions vers le bon serveur, il convient de configurer l'url du serveur d'impression. 

  1. Depuis l'interface Google Workspace Admin, dans le menu de gauche, cliquez sur Applications > Applications Web et mobiles.

  2. Dans l'interface Applications, cliquez sur l'application Watchdoc Print Client précédemment ajoutée.

  3. Dans l'interfaceWatchdoc Print Client, section Configurations gérées, cliquez sur Ajouter une configuration gérée :

  1. Dans l'interface Ajouter une configuration gérée, saisissez :

    • le nom de la configuration

    • l'adresse url du serveur d'impression vers lequel envoyer les travaux :

  2. Cliquez sur le bouton Enregistrer.

    La configuration apparaît dans la liste des configurations gérées.

  3. Pour appliquer la configuration aux utilisateurs, cliquez sur le lien Appliquer une configuration située au-dessus de la liste des configurations gérées :

    • cliquez sur le lien Modifier l'accès si vous souhaitez appliquer la configuration gérée à d'autres utilisateurs ;

    • autrement, vérifiez les informations affichées et cliquez sur Enregistrer si vous souhaitez appliquer la configuration gérée aux utilisateurs mentionnés dans le bandeau supérieur :
