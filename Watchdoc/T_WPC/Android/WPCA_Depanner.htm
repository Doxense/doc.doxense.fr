Télécharger les journaux (logs)

Pour diagnostiquer un dysfonctionnement de l'application, demandez à l'utilisateur de vous envoyer les journaux qu'il aura téléchargés directement depuis l'application :

  1. Quand le problème survient, cliquez sur l'icone Paramètres dans le bandeau bas de l'application :

  2. cliquez sur le bouton Télécharger les logs ;

     

  3. parmi les outils de messagerie proposés, sélectionnez celui avec lequel vous souhaitez envoyer les logs et complétez l'adresse de votre destinataire :


    N.B. : vous pouvez aussi envoyer des fichiers de logs dans un espace de stockage partagé

  4. confirmez l'envoi du fichier

    è le message "L'importation a commencé" confirme l'envoi des logs au destinataire.

L'url de découverte ou le certificat est mal configuré dans l'EMM. 

Vérifiez les données saisies pour les clés de configuration Serveur de découverte et Certificat dans votre EMM (cf. WPC for Android - Ajouter et configurer l'application dans l'EMM Intune).

N.B. : pour être sûr de disposer du certificat de la Print API sous forme d'une chaîne de caractères sans espaces ni retours-chariots, copiez-le en suivant la procédure décrite das les Prérequis (cf. Ajouter et configurer l'application, dans l'EMM Intune).

Si le certificat a déjà été configuré avec des erreurs, supprimez-le puis recréez-le.
