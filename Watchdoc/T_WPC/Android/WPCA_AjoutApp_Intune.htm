Watchdoc Print Client for Android - Ajouter et configurer l'application dans l'EMM Intune

Principe

En tant qu'administrateur, lorsque vous fournissez votre ID d'organisation, Doxense l'ajoute à la liste des organisations / entreprises autorisées à utiliser l'application Watchdoc Print Client for Android. Cette application est mise à votre disposition dans le store Google Play. 

 

Dès que vous avez accès à l'application dans Google Play, vous pouvez la mettre à disposition de vos utilisateurs de deux manières différentes :

  1. soit en la configurant dans votre EMM Microsoft Intune de sorte qu'elle soit mise à disposition de vos utilisateurs depuis leur environnement Intune professionnel ;

  2. soit en fournissant à vos utilisateurs l'application sous forme d'un fichier .apk. Chaque utilisateur doit installer le fichier .apk sur son smartphone, puis doit le configurer à l'aide des paramètres que vous fournissez.


Dans l'EMM Microsoft Intune, comme pour l'apk, les paramètres à configurer pour l'application sont :

  • l'url du "serveur de découverte" qui permet à l'application de communiquer avec le serveur d'impression Watchdoc ;

  • le certificat de la Print API (fichier au format .der), garant de la sécurité de l'application.

Prérequis

Pour configurer l'application Watchdoc Print Client for Android dans Intune, vous devez disposer du certificat print_api_cert.der au format base 64.
Ce certificat doit être exporté au format .der depuis Watchdoc puis modifié en fichier .txt :

Configurer Watchdoc Print Client for Android dans InTune

Accéder à l'interface de configuration

Pour que l'application Watchdoc Print Client for Android puisse communiquer avec le serveur Watchdoc, il convient d'associer l'application Watchdoc Print Client for Android à l'url du serveur d'impression dans l'EMM et de l'associer au certificat :

  1. rendez-vous dans l'interface d'administration Centre d'administration Microsoft Intune ;

  2. dans le menu de gauche, cliquez sur Applications ;

  3. dans l'interface Applications|Vue d'ensemble, cliquez sur la tuile Android ;

  4. dans l'interface Android|Android applications, cliquez sur le bouton Créer ;

  5. dans la fenêtre de droite, sélectionnez le type d'application Application Google Play géré, puis cliquez sur Sélectionner ;

     

  6. dans Managed Google Play, cherchez et sélectionnez l'application Watchdoc Print Client for Android :



Configurer l'url de découverte et configurer le certificat WPC

Une fois l'application ajoutée,

  1. dans le menu Android applications, cliquez sur Configuration :



     

  2. dans la liste des applications, sélectionnez l'application Watchdoc Print Client for Android ;

  3. dans l'interface de configuration de l'application, cliquez sur l'entrée de menu Propriétés ;

  4. dans la section Paramètres de configuration, en Format des paramètres, sélectionnez Utiliser le concepteur de configuration ;

  5. complétez ensuite la section Paramètres, en ajoutant 2 clés de configuration :

    1. le serveur Watchdoc

      • saisissez la valeur "Serveur de découverte" ;

      • sélectionnez la valeur "string" ;

      • dans le champ "Valeur", saisissez l'adresse url du serveur Watchdoc, suivie du port d'accès et de la mention "/discover" ;

      • dans le champ "Description", fournissez une description de la clé de configuration si nécessaire ;

    2. le certificat

      • saisissez la valeur "Certificat' ;

      • sélectionnez la valeur "String" ;

      • dans le champ 'Valeur", saisissez la chaîne de caractères correspondant au certificat préalablement téléchargé depuis Watchdoc ;

      • dans le champ "Description", fournissez une description de la clé de configuration si nécessaire (Exemple : certificat de la Print API) ;

  6. dans la section Paramètres de configuration, en Format des paramètres, sélectionnez Entrer des données JSON ;


  7. Dans la section Applications connectées, cliquez sur Activé pour autoriser vos utilisateurs à se connecter à l'application à l'aide de leur profil personnel ou professionnel en fonction de la politique d'accès aux applications en vigueur dans votre organisation.

  8. cliquez sur Enregistrer pour valider la configuration de l'application.
