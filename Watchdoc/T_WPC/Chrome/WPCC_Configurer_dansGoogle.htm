Watchdoc Print Client for Chrome® - Configurer

Accéder à l'interface de configuration

Une fois la fonction WPC for Chrome activée dans Watchdoc, installez l'extension dans votre environnement Chrome® :

  1. accédez à l'outil d'administration de Google Admin (https://Admin.google.com).

  2. dans le menu, cliquez sur Devices > Chrome > Apps&Extension ;

     

Ajouter l'extension

  1. avec le moteur de recherche, cherchez l'organisation Doxense, puis cliquez dessus dans le menu de gauche pour la déployer et voir toutes les applications qui s'y rapportent :

  2. dans la liste des applications, parmi les outils figurant en bas à droite, cliquez sur le bouton pour ajouter une extension :


  1. dans l'interface Add the Chrome app or extension by ID, saisissez :

    • soit l'identifiant, dans le : champ Extension ID,

    • soit l'URL dans le champ From a custom URL,

  2. validez en cliquant sur :

     

  3. l'extension apparaît dans les sources disponibles :

     

     

  4. cliquez dessus et complétez le champ Policy for extensions : pour affiner la stratégie de l'extension, ajoutez le code suivant :

    {"settings":{"discoveryUrls":["http://10.10.0.173:5753/api/1.0/print/discover"]}}

     

  1. Sélectionnez ensuite le mode d'installation :

    • Force install : sélectionnez ce mode pour installer l'extension par défaut sur tous les postes Chromebook® ou navigateurs Chrome® gérés par votre organisation ;

    • Allow install : sélectionnez ce mode pour proposer cette extension aux utilisateurs de vos postes Chromebook® ou navigateurs Chrome® gérés par votre organisation en les laissant libres de l'activer ou non ;

    • Bloc : sélectionnez ce mode pour bloquer l'utilisation de cette extension sur les postes Chromebook® ou navigateurs Chrome® gérés par votre organisation :


  2. Cliquez ensuite sur Sécurité > Contrôle des accès et des données > Commandes des API.

  3. S'il y en a plusieurs, recherchez l'unité organisationnelle que vous souhaitez modifier.

  4. Dans l'interface Affichage des paramètres pour les utilisateurs de [unité_organisationelle], pour le paramètre "Applications tierces non configurées", sélectionnez l'option "autoriser les utilisateurs à accéder aux applications tierces qui ne demandent que les informations de base nécessaires pour la fonctionnalité Se connecter avec Google" :



  1. Validez le paramétrage en cliquant sur .

  2. Testez l'extension à partir d'un Chromebook® ou d'un navigateur Chrome®.
