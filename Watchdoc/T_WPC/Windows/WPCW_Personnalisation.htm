Watchdoc Print Client for Windows - Personnaliser graphiquement

Principe

Comme pour la plupart des WES et WEScan, l'interface graphique de Watchdoc Print Client for Windows peut être personnalisée à l'aide de couleurs, d'images et logos propres à votre organisation.

Procédure

Tester la personnalisation des images

Avant de modifier les images par défaut de WPC for Windows sur le serveur Watchdoc (avec impact sur l'ensemble des postes clients), vous pouvez mener un test de personnalisation depuis un poste client :

  1. accédez en tant qu'administrateur à un poste client sur lequel est installé WPC for Windows ;

  2. à l'aide de l'explorateur, rendez-vous dans le dossier images de WPC (C:\Programmes\Doxense\Watchdoc Print Client\bin\assets\images\) ;

  3. dans ce dossier, créez un dossier nommé test_customization ;

  4. dans ce dossier, créez les images personnalisées correspondant aux interfaces WPC for Windows. Pour cela, respectez les dimensions des images d'origine :

    • background.png : 211pt x 232 pt

    • filigran.png : 216 pt x 100.5 pt

    • logo.png : 330 pt x 36.75 pt

    • user_profile.png : 180,5 pt x 164 pt


     

  5. fermez WPC et redémarrez le service WPC sur le poste client ;

  6. lancez WPC for Windows pour vérifier l'affichage des images personnalisées :


     

  1. Si nécessaire, retouchez les images pour qu'elles soient parfaitement adaptées à vos besoins.

     

èDès que les images personnalisées sont satisfaisantes, rendez-vous sur le serveur master Watchdoc pour finaliser la personnalisation. Les images testées devront y être disponibles (par copier/coller ou par enregistrement dans un dossier accessible depuis le master).

Personnaliser les images sur le serveur

  1. Accédez en tant qu'administrateur au serveur master ;

  2. rendez-vous dans le dossier contenant les images de WPC (par défaut : C:\Program Files\Doxense\Watchdoc\Images\WindowsClient) ;

  3. dans ce dossier, créez un dossier dédié aux images personnalisées (par exemple Custom) ;

  4. dans le dossier Custom, enregistrez vos images personnalisées (précédemment testées sur le poste client) :


 

Configurer la personnalisation des images

  1. Accédez en tant qu'administrateur à l'interface d'administration de Watchdoc ;

  2. depuis le Menu principal, section Configuration, cliquez sur Configuration avancée > Configuration clientes ;

  3. dans la liste, sélectionnez Client Windows ;

  4. dans la section Configuration du Windows Client, indiquez le chemin vers le dossier contenant les images personnalisées (dans notre exemple : C:\Program Files\Doxense\Watchdoc\Images\WindowsClient\Custom) :

 

Configurer la personnalisation de la couleur

  1. Toujours dans l'interface Configuration Client Windows, section Interface utilisateur, paramètre Couleur..., indiquez la valeur hexadécimale de la couleur qui s'applique :

    • au fil d'Ariane,

    • à l'information de localisation

    • aux boutons de l'interface :

 

  1. Cliquez sur le bouton pour valider la personnalisation de WPC for Windows.

  2. Vérifiez dans WPC for Windows que ces images ont bien été prises en compte. Il est possible d'avoir à fermer, puis relancer l'application pour constater la modification.
