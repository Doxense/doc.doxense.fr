WPC for Windows - Comptabiliser l'impression directe

Principe

Dans certains contextes, l'activité d'impression peut être gênée par des contraintes techniques. Par exemple, quand les postes de travail sont éloignés géographiquement du serveur d'impression, le délai d'acheminement des spools entre les postes et le serveur peut alors ralentir la productivité des employés.

 

Dans ce cas, l'impression directe de WPC for Windows permet de contourner ces contraintes techniques spécifiques. Les utilisateurs peuvent ainsi imprimer sans solliciter le serveur Watchdoc.

 

Mais pour que les données statistiques de ces périphériques d'impression puissent être collectées, il est nécessaire de configurer dans Watchdoc la file qui servira à l'impression directe. Comme il s'agit d'un mode dégradé et que Watchdoc n'est pas sollicité, les données statistiques ne sont pas aussi détaillées que celles qui sont habituellement collectées par Watchdoc. Néanmoins, il est possible de collecter a minima des informations fournies par le spooler MS Windows (printinfo2 : titre du document, nombre de pages, couleur (couleur ou monochrome) et utilisateur).

Configuration

  1. Accédez à l'interface d'administration WSC en tant qu'administrateur ;

  2. depuis WSC, ajoutez un pilote (s'il n'existe pas) et créez la file physique correspondant au périphérique d'impression utilisé pour l'impression directe ;

  3. accédez à l'interface d'administration Watchdoc en tant qu'administrateur ;

  4. depuis le Menu Principal, cliquez sur Files d'impression, emplacements, groupes de files & pools ;

  5. dans la liste Files d'impression, cliquez sur Emplacements ;

  6. dans la liste des emplacements, ajoutez un emplacement dédié aux postes qui utiliseront l’impression directe ;

  7. dans la liste Files d’impression, cliquez sur Périphériques d'impression > Périphériques non-contrôlés ;

  8. dans la liste des périphériques non-contrôlés, cliquez sur le bouton Ajouter pour soumettre la file au contrôle de Watchdoc ;

  9. dans la section Informations générales de cette file,

    • précisez l'Emplacement dédié aux postes qui utilisent l'impression directe ;

    • pour le paramètre Tags, saisissez les valeurs remote et tcpip ;

       

  10. vérifiez dans la configuration de WPC for Windows que l’impression directe est activée et configurée (cf. Configurer WPC for Windows) :


  11. installez WPC for Windows sur les postes de travail (cf. Installer WPC for Windows sur le poste utilisateur) ;

  12. informez les utilisateurs sur les conditions d’utilisation de la file d’impression directe.

     

    è Une fois cette file configurée, lorsqu’un utilisateur opte pour l’impression directe, les données du spooler Windows sont transmises au serveur master de Watchdoc et consultables sous forme de bilans.
