Watchdoc Print Client for Windows - Désinstaller

Procédure

Désinstallation en masse

La désinstallation en masse à l'aide peut être réalisée :

  • en stratégie de groupe, sous GPO :

    1. rendez-vous dans la console Gestion des stratégies de groupe ;

    2. rendez-vous dans Configuration ordinateur > Stratégies > Paramètres logiciels > Installation de logiciel ;

    3. sélectionnez Watchdoc Print Client for Windows et cliquez-droit ;

    4. sélectionnez Toutes les tâches > Supprimer ;

    5. choisissez l'option : "Désinstaller immédiatement le logiciel des utilisateurs et des ordinateurs".

    6. ou saisissez la ligne de commande : msiexec.exe /x "WatchdocPrintClient-%VERSION%.msi" /qn /norestart (cf. Commandes Windows msiexec pour plus de détails).

     

  • sous Intune, le package .msi peut être désinstallé à l'aide de l'outil Microsoft Endpoint Manager admin center.

    1. connectez-vous au centre d'administration Microsoft Intune ;

    2. rendez-vous dans Applications > Windows ;

    3. sélectionnezWatchdoc Print Client for Windows ;

    4. rendez-vous dans Propriétés, puis à côté de la section Affectations, cliquez sur Modifier ;

    5. sous la colonne Exclus ou Désinstaller, ajoutez les groupes d'utilisateurs ou d'appareils concernés.
      Note : si vous voulez le supprimer pour tout les utilisateurs, retirez les groupes de la section "Requis" et mettez-les dans "Désinstaller".

    6. Enregistrez.

      7. èIntune envoie l'ordre de désinstallation lors de la prochaine vérification des postes de travail.
