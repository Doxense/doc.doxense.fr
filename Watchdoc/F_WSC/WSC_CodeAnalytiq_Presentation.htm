Console de Supervision (WSC) - Codes analytiques : présentation

Principe

Le code analytique est un code correspondant à un compte spécifique sur lequel les impressions demandées par les utilisateurs peuvent être imputées. Ce code permet donc la ventilation des coûts d'impression facilitant une comptabilité analytique de ce poste.

 

Pour activer la fonction Codes analytiques, il est nécessaire de disposer d'une liste des codes (représentant des services, unités, départements, clients, etc.) auxquels il faut imputer les impressions. Par exemple :

  • les codes correspondent aux différents services d'une entreprise, de sorte que les impressions de Carole Coutant sont imputées au Service Commercial et celles de Stéphane Sol sont imputées au Support ;

  • ou les codes correspondent aux clients d'un avocat, de sorte qu'Aude, avocate, imprime les documents en les imputant au code du client qu'elle défend afin de refacturer à ce client les coûts d'impression relatifs à son dossier.

Etapes

L'activation de la fonction Codes analytiques se paramètre :

  1. accessoirement, la base de données affectée à la fonction (configurée automatiquement et par défaut dans Watchdoc) peut être modifiée ;

  2. dans la console de supervision Watchdoc (WSC) : les codes analytiques sont saisis et hiérarchisés (si nécessaire) ;

  3. dans le profil WES : la fonction est activée pour permettre aux utilisateurs de choisir ou de saisir leur code analytique depuis l'écran du périphérique.

Contraintes de compatibilité

La gestion des codes analytique est compatible avec les périphériques des marques et technologies suivantes :

Marque Technologie

WES compatible

Gamme 4e/8 et sup. WES V3
EIP 2.X/3.X WES V3
