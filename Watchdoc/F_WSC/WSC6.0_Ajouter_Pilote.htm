WSC Console de Supervision v6.0 - Charger un pilote (driver)

  1. Accédez à la Console de Supervision (WSC) en tant qu'administrateur ;

  2. depuis le Menu Principal, cliquez sur Gestion des pilotes :

 

  1. Dans la liste d'actions, cliquez sur le bouton Ajouter un pilote d'impression qui est dans le bloc à droite de l'écran.
    L'interface d'ajout de pilote s'affiche :

 

  1. Sélectionnez le pilote que vous avez téléchargé sur le site du constructeur.
    Le pilote doit être de type 3.

→ Le pilote est reconnu par Watchdoc.

  1. Désactivez l'option Signer le package.

  2. Cliquez sur Suivant :

 

  1. Le pilote est analysé par Watchdoc :
    .

 

  1. Cliquez sur Suivant.

→ Par défaut, Watchdoc propose de créer une file d'impression universelle associée au pilote. Cette file permettra d'imprimer avec ce pilote sur tous vos périphériques d'impression.

  1. Créez cette file en la nommant Impression sécurisée.

 

→ Watchdoc déploie le pilote sur le serveur d'impression et crée la file universelle (pour plus d'informations, cf.Configurer une file universelle.)


.

 

→ Vous pouvez ensuite Activer l'interface WES
