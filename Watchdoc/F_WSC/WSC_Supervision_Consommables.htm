Console de Supervision (WSC) - Superviser les consommables

Principe

La Console de Supervision permet d'obtenir des vues détaillées et affinées des consommables des files d'impression contrôlées par Watchdoc et notamment les consommables ayant atteint un faible niveau (20% par défaut).

Accéder à l'interface de supervision

Depuis le Menu principal de WSC, cliquez sur bouton Consommables :

Vous accédez à l'interface Consommables, dans laquelle s'affiche la liste des files dont les niveaux de consommables sont critiques (faible niveau de toner, tambour ou autres en-deça de 20%) :

Dans cette liste, vous pouvez accéder à différentes vues en cliquant sur les différents onglets et filtrer les résultats en modifiant le seuil (entre 1 et 100%).

Consulter les consommables d'une file

Depuis la liste des files d'impression :

  1. sélectionnez la file dont vous souhaitez consulter le niveau de consommables;

  2. cliquez sur les informations affichées dans les colonne pour consulter le détail des niveaux relevés :

  • Toner : le niveau des toners restants et les couleurs concernées ;

  • Tambour : le niveau des tambours restants et les couleurs concernées ;

  • Autres : les autres niveaux disponibles (kit de fusion, courroie de transfert, etc.) :

  1. cliquez sur :

  • le nom du serveur pour accéder à l'interface d'administration du serveur ;

  • le nom du périphérique pour accéder à l'interface d'administration Watchdoc® de ce périphérique ;

  • vous accédez à l'interface d'administration de Watchdoc® : authentifiez-vous comme administrateur pour y consulter le détail des informations relatives à la file.

  • l'adresse IP du périphérique pour accéder à l'interface d'administration du périphérique lui-même.

Modifier le seuil

Si le seuil par défaut qui déclenche l'envoi du rapport ne vous convient pas, vous pouvez le modifier :

  1. depuis le Menu Principal de WSC, cliquez sur Configuration avancée, puis Service ;

  2. dans l'interface Configuration Système, rendez-vous dans la section Divers ;

  3. dans le champ Seuil, saisissez le niveau de consommables, en pourcentage, en deça duquel WSC envoie un rapport par mail :

  4. Cliquez sur Valider pour enregistrer la configuration.

  5. Pour activer l'envoi d'un rapport, reportez-vous à la procédure de configuration.
