WSC Console de Supervision - Pilotes (drivers) - Présentation

Créer les pilotes dans WSC

WSC vous permet de référencer le ou les pilotes (drivers) correspondants aux périphériques d'impression de votre parc. Pour les référencer dans WSC, il est nécessaire d'avoir préalablement téléchargé et enregistré les packages d'installation dans un environnement de travail accessible depuis WSC.

 

WSC v6.1 comporte un module de sécurisation des pilotes d'impression. A partir de cette version, seuls les pilotes certifiés par Microsoftc® avec une signature WHQL Signature numérique attribuée aux packages de pilotes répondant aux exigences des tests HLK (Windows Hardware Lab Kit) par Microsoft pour garantir la sécurité du pilote. (Source : https://learn.microsoft.com) valide peuvent être installés sur la console de Supervision.
Par ailleurs, il est nécessaire d'utiliser l'utilitaire Package extractor pour ajouter des pilotes conformes dans WSC v6.1. 

 

N.B. : nov. 2024 : une anomalie empêche l'installation des pilotes Ricoh et Epson à l'aide de DriverPackageExtractor. Il convient donc d'installer ces pilotes via le gestionnaire d'impression, GPO ou à l'aide d'Intune® en attendant une correction de l'utilitaire. 

Mettre à jour les pilotes dans WSC

La procédure de chargement des pilotes ayant changé en v. 6.1, lors d'une mise à jour de WSC v6.0 (ou antérieure) vers v6.1, il est nécessaire de supprimer les pilotes et de les charger de nouveau en respectant la procédure (cf. WSC v6.1 - Charger un pilote).
