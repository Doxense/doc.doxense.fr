WSC Console de Supervision - Pilotes (drivers) - Présentation

Créer les pilotes dans WSC

WSC vous permet de référencer le ou les pilotes (drivers) correspondant aux périphériques d'impression de votre parc. Pour les référencer dans WSC, il est nécessaire d'avoir préalablement téléchargé et enregistré les packages d'installation dans un environnement de travail accessible depuis WSC. Nous vous recommandons de vous procurer ces packages sur les sites des fabricants pour en garantir la fiabilité.

 

WSC v6.1 comporte un module de sécurisation des pilotes d'impression. A partir de cette version, les pilotes certifiés par Microsoft® avec une signature WHQL Signature numérique attribuée aux packages de pilotes répondant aux exigences des tests HLK (Windows Hardware Lab Kit) par Microsoft pour garantir la sécurité du pilote. (Source : https://learn.microsoft.com) valide peuvent être installés sur la console de Supervision à l'aide de l'utilitaire Driver Package Extractor.

 

Il existe cependant deux exceptions : les pilotes des marques Ricoh et Epson signés à l'aide d'un certificat WHQL Signature numérique attribuée aux packages de pilotes répondant aux exigences des tests HLK (Windows Hardware Lab Kit) par Microsoft pour garantir la sécurité du pilote. (Source : https://learn.microsoft.com) génèrent un message informant que le pilote doit préalablement être signé par un certificat interne (certificat du domaine client ou par Doxense). Donc, si vous souhaitez installer l'un de ces pilotes :

  • demandez à votre organisation de signer le pilote ;

  • ou, si votre organisation n'est pas en mesure de signer le pilote, contactez le Support Doxense via Connect afin de demander la fourniture d'un certificat.

Dans ces deux cas, le nom de l'émetteur du certificat (issuer) sera indiqué dans l'assistant d'installation du pilote.

Mettre à jour les pilotes dans WSC

La procédure de chargement des pilotes ayant changé en v. 6.1, lors d'une mise à jour de WSC v6.0 (ou antérieure) vers v6.1, il est nécessaire de supprimer les pilotes installés et de les charger de nouveau en respectant la procédure (cf. WSC v6.1 - Charger un pilote).
