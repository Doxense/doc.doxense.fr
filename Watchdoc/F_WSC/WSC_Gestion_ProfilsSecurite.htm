Console de Supervision (WSC) - Créer et configurer les profils de sécurité

Principe

La Console de Supervision peut être sécurisée à l'aide de profils de sécurité.
Ces profils permettent d'attribuer des droits d'accès et de gestion spécifiques à différents groupes d'utilisateurs.

Accéder à l'interface de configuration

  1. Depuis le Menu principal de WSC, cliquez sur Configuration avancée ;

  2. Dans l'interface Configuration avancée, cliquez sur Profils de sécurité :

     

  3. Si aucun profil n'existe, vous accédez directement à la section Propriétés du profil à créer. Complétez les champs pour créer le profil.
    Si un profil existe déja, vous accédez à la liste des profils. Dans la liste cliquez sur le bouton Créer un profil puis complétez les champs.

Configurer le profil

  1. Dans les Propriétés,

    • Nom du profil : saisissez un nom attribué à ce profil d'administration (par exemple Super Admin, Admin Badges ou Opérateurs Badges) en fonction des habilitations du profil ;

    • Description : complétez par une description des habilitations, si nécessaire ;

    • Groupes : saisissez le nom d'un groupe de l'annuaire ou le nom d'un utilisateur spécifique (préfixé par @).

  1. Droits : dans la grille, précisez les droits qui sont autorisés ou refusés au profil configuré. 
    Par défaut, un droit est refusé.
    Si un utilisateur appartient à plusieurs groupes (et donc à plusieurs profils), la flèche permet de parcourir tous les profils pour lui accorder le droit, si ce droit est accordé à un autre profil auquel il appartient.

     

  1. Cliquez sur le bouton pour enregistrer le profil.

Modifier un profil

Pour modifier un profil

  1. depuis l'interface Profils de sécurité, cliquez sur le bouton Editer d'un profil ;

  2. dans la liste des droits, redéfinissez les droits accordés au autorisés au profil édité ;

  3. Cliquez sur le bouton pour enregistrer les modifications du profil.
