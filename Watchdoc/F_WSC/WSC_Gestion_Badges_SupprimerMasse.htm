Console de supervision (WSC) - Supprimer des badges

Supprimer un badge

Pour supprimer un badge,

  1. dans la liste Badges, cliquez sur le bouton correspondant au badge à supprimer ;

  2. dans le menu affiché, sélectionnez Supprimer ce badge :



  1. dans la boite Suppression d'un badge, confirmez votre action.

Supprimer plusieurs badges

Pour supprimer plusieurs badges simultanément :

  1. dans la liste des badges, cochez les cases pour sélectionner les badges à supprimer ;

  2. cliquez sur le bouton Supprimer la sélection situé en base de la liste ;

  3. dans la boîte Suppression de badges, validez votre demande de suppression.

 

Supprimer des badges en masse

La Console de Supervision dispose d'un outil permettant de supprimer des badges en masse à partir d'une liste de badges extraits d'une base de données tierce.

Pour supprimer une liste de badges dans la Console de supervision, il convient donc d'avoir, au préalable, extrait les badges dans un fichier CSV.

Pour importer la liste de badges à supprimer depuis le fichier CSV :

  1. depuis l'interface Gestion des badges, cliquez sur le bouton

  2. depuis la fenêtre Suppression de masse depuis fichier CSV, cliquez sur le bouton pour rechercher le fichier d'import dans votre espace de travail ;

  3. sélectionnez le fichier .csv à importer ;


  4. sélectionnez le format du fichier .csv ;

  • case "Ne rien importer si des erreurs sont présentes" :

    • cochez-la pour ne supprimer les badges qu'à la condition qu'il n'existe aucune erreur dans le fichier d'import ;

    • décochez la case pour procéder à la suppression, même en cas d'erreur dans le fichier d'import. 

Les erreurs survenues au cours de l'import sont signalées dans le rapport d'import affiché au terme de l'opération. 

  1. cliquez sur pour procéder à la suppression des badges.

→ Au terme de l'opération, un message d'information est affiché, soit pour vous informer de son succès, soit pour vous signaler les erreurs survenues.

 

→ Le message d'erreur est générique si vous avez coché la case "Ne pas importer..." ou précise la (les) ligne(s) erronée(s) si vous avez décoché la case :

Le message d'erreur signale soit une lacune dans le fichier d'import (valeur manquante dans une colonne), soit une incompatibilité entre le format choisi dans l'interface et le format du fichier.

Supprimer des badges inutilisés

Pour supprimer des badges inutilisés :

  1. depuis l'interface Gestion des badges, cliquez sur l'onglet Rechercher les badges non utilisés ;

 

  1. dans l'interface de recherche des badges non-utilisés, saisissez une date ou sélectionnez la date dans le calendrier affiché sous le champ ;

     

  1. après avoir indiqué la date de début de recherche, cliquez sur le bouton ;

  2. le résultat de recherche s'affiche dans la liste ;

  3. dans la liste, sélectionnez le badge à supprimer, puis cliquez sur le bouton pour accéder au menu contextuel ;

  4. dans le menu contextuel, cliquez sur le bouton :
