Première utilisation de Watchdoc

Vous souhaitez imprimer, mais une nouvelle application se trouve sur votre copieur ? Bienvenue sur Watchdoc, votre nouveau logiciel de gestion des impressions et de numérisation. 

 

Dans ce tutoriel, nous vous accompagnons pas à pas dans l'impression de vos documents, rassurez-vous, c'est très facile. 

 

Tutoriel vidéo

Obtenir votre code d'impression

  1. Rendez-vous dans la page "Mon compte" et authentifiez-vous.

  2. Dans l'interface, cliquez sur l'onglet "Mes codes et badges".

  3. Les codes d'accès (PUK et/ou PIN) sont affichés dans l'interface, mais masqués.

  4. cliquez sur le bouton pour les afficher en clair :

 

Il est également possible de recevoir votre code par e-mail.

S'identifier

  1. Réveiller l'équipement

     

    Le périphérique semble éteint. 

    S'il ne se réveille automatiquement pas lorsque vous approchez, cliquez sur une touche allumée ou tapotez l'écran.

 

  1. S'identifier

     

    Passez votre badge ou votre pastille sur le lecteur ou saisissez votre code d'identification (PIN/PUK ou login/mot de passe) sur l'écran pour accéder aux icones.


    Authentification par badge
    Authentification par code

     

  2. Première utilisation avec un badge

     

    Lors de la première utilisation du badge, vous devez l'associer à votre compte : passez votre badge sur le lecteur puis saisissez votre code d'identification (PIN/PUK ou login/mot de passe).

     

    Pour les utilisations suivantes, seul le badge sera nécessaire pour l'identification.


    Message d'invitation à associer votre badge

    Authentification par code ou login/mot de passe


Se déconnecter

Passez votre badge sur le lecteur ou appuyez sur le bouton de déconnexion de l'écran.
