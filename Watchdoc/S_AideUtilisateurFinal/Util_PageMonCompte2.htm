Watchdoc - Utilisateur - Consulter la page Mon compte Nouvelle version

Présentation

La page "Mon compte" est l'interface web de Watchdoc dédiée à l'utilisation. Cette interface est déclinée en deux versions : l'ancienne et la nouvelle, depuis Watchdoc v. 6.1.1.5854.

Quelle que soit la version utilisée, son url est généralement communiquée par votre référent Watchdoc à l'ensemble des utilisateurs.


Configurée par l'administrateur, elle peut proposer des usages variés. Elle peut donc permettre :

  • de consulter l'historique de vos impressions ;

  • de consulter les statistiques de vos impressions et des impressions de votre équipe ;

  • de consulter le détail de vos quotas d'impression ;

  • de consulter les documents archivés ;

  • de consulter vos données d'identification telles que vos codes d'identification et badges.

Consulter ses codes et badges

Pour consulter la liste de votre (ou vos) code(s) d'identification et/ou badges enregistrés :

  1. depuis l'interface de déblocage de vos documents, cliquez sur Mon compte ;

  2. cliquez sur l'onglet Mes codes et badges ;

    • dans la liste Vos codes d'identification figurent plus ou moins de codes (PIN, PUK, d'impression) en fonction de la configuration appliquée. Si le code est masqué, cliquez sur le bouton Voir pour afficher le code ;

    • dans la liste Liste de vos badges enregistrés figurent le ou les badges que vous êtes en mesure d'utiliser pour vous identifier sur un WES :

      Les badges peuvent avoir plusieurs statuts :

      • actif : le badge est utilisable

      • supprimé :

      • suspendu :

      • ...



Consulter ses statistiques

Pour consulter les statistiques de votre activité d'impression :

  1. depuis l'interface de déblocage de vos documents, cliquez sur Mes statistiques ;

  2. cliquez sur l'onglet Mes Statistiques : vous accédez à une interface qui affichent vos données d'usage de Watchdoc et l'impact environnemental qui en découle :

Consulter ses impressions - Historique des derniers documents

Pour consulter l'historique de vos dernières impressions :

  1. depuis l'interface de déblocage de vos documents, cliquez sur Mon compte ;

  2. cliquez sur l'onglet Mes impressions ;

  3. dans la liste Historique de vos derniers documents figurent les documents dont vous avez demandé l'impression. Le nombre de document varie en fonction de la configuration définie par l'administrateur.

  4. la liste comporte différents indicateurs :

    • dans la colonne Nom du document figure un pictogramme indiquant le type/format du document (.pdf, .txt,, .doc, .xls, .jpg, etc.) ;

    • à droite du document figurent des pictogrammes indiquant le traitement qui lui a été appliqué :

      • indique un document qui a été imprimé ;

      • indique un document qui a été supprimé (soit par vous-même, soit automatiquement en raison d'une politique d'impression spécifique) ;

      • indique un document expiré, c'est-à-dire supprimé après être resté trop longtemps dans la liste des travaux en attente sans avoir été explicitement imprimé ou supprimé ;

    • dans la colonne Pages figurent des critères d'impression :

      • le nombre de pages,

      • le format du document (A4, A3, letter, etc.)

      • le mode couleur (couleur ou noir et blanc ),

      • le mode recto ou recto version

    • dans la colonne Imprimante figure le nom de l'imprimante chargée de traiter le document ;

    • dans la colonne Date figure l'heure du traitement de l'impression pourl le jour courant ou la date du traitement pour les jours précédents.

Consulter ses quotas

Pour consulter l'état de ses quotas et opérations relatives aux quotas:

 

  1. depuis l'interface de déblocage de vos documents, cliquez sur Mon compte ;

  2. cliquez sur l'onglet Mes Quotas ;

  3. dans la liste des quotas activés, vous pouvez consulter

    • l'état du quota

    • le solde utilisateur

    • le coût de la dernière opération.

Consulter ses archives

Si la fonction d'archivage est autorisée et que vous l'avez utilisée, vous pouvez consulter la liste des documents archivés. 

Pour consulter la liste de documents archivés,

  1. depuis l'interface de déblocage de vos documents, cliquez sur Mon compte ;

  2. cliquez sur l'onglet Mes archives :


Vous pouvez cocher les cases des documents figurant dans cette liste pour lui appliquer des traitements divers :

  • supprimer le document archivé ;

  • rendre le document archivé temporaire ;

  • rendre le document archivé permanent ;

  • imprimer le document archivé.
