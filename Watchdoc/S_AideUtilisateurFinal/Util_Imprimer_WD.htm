Imprimer avec Watchdoc

Si c'est votre première impression, référez-vous plutôt à la page "Première utilisation".

Tutoriel vidéo

Accéder au WES

  1. après avoir lancé votre impression depuis votre PC (via votre file d'impression), sélectionnez votre imprimante.

  2. Rendez-vous ensuite sur l'imprimante, identifiez-vous sur le WES (interface embarquée sur votre imprimante).

  3. Une fois authentifié, cliquez sur le bouton

  4. Vous arrivez sur cette interface :


 

  1. Cochez les cases pour sélectionner vos documents.

  2. Sélectionnez "Imprimer" pour imprimer les documents cochés

  3. ou cliquez sur "Supprimer" pour supprimer les documents cochés.

 

: ces icônes informent sur le type de document, le mode d'impression recto ou recto-verso et la définition couleur ou noir&blanc ;

Ce bouton permet d'activer une barre d'outils pour chaque document :

  • voir en détail les documents

  • modifier les documents

  • supprimer les pages superflues

  • supprimer le document.

Modifier les pages*

  1. Cliquez sur pour consulter les modifications possibles :

    • Noir & blanc ou couleur

    • Verso ou recto-verso

    • Nombre d'exemplaires

  1. Effectuez toutes les modifications souhaitées.

  2. Cliquez sur "Imprimer" pour lancer l'impression.

Supprimer des pages*

  1. Cliquez sur pour accéder à l'interface de suppression.

  2. Parcourez le document.

  3. Cliquez sur les pages à supprimer : elles sont alors barrées de rouge.

  4. Cliquez sur le bouton pour pour imprimer le reste des pages.

 

* Attention : ces fonctionnalités dépendent de la marque de l'imprimante et de la version du WES. Veuillez vous rapprocher de votre service informatique pour vérifier que vous disposez bien de ces fonctions.
