Table billingcodes

 

Description

Cette table contient les codes analytiques et utilise l'imbrication d'ensembles (nested set model).

La modification des données de cette table risque d'entraîner une impossibilité d'utiliser la fonction "Codes analytiques". Ces derniers doivent être gérés depuis la Console de Supervision (WSC).
