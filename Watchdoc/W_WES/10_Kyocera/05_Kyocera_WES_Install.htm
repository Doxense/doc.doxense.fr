WES Kyocera - Installer le WES sur la file

Présentation de la section WES

Une fois le profil WES activé sur la file, dans l'interface Propriétés de la file apparaît la section WES Kyocera HyPAS.
Cette section comporte plusieurs boutons :

  • Accéder à l'interface WEB du périphérique : raccourci vers le site web d'administration interne du périphérique ;

  • Périphériques WES : donne accès à la liste des périphériques équipés d'un WES dans Watchdoc ;

  • Configurer: permet à Watchdoc d'installer le WES sur le périphérique (peut prendre 30 sec.) ;

  • Activer / Désactiver les logs : permet d'activer les logs spécifiques au WES en vue d'une analyse ;

  • Télécharger les logs : permet de télécharger les logs spécifiques au WES (s'ils ont été activés) en vue d'un diagnostic ;

  • Désinstaller : permet à Watchdoc de désinstaller le WES sur le périphérique. Une fois le WES désinstallé, il convient de redémarrer le périphérique ;

  • Editer la configuration : permet d'accéder à la configuration du WES sur la file.

  • Page de configuration du WES : permet d'accéder à l'interface de configuration du WES dans Watchdoc.

Procédure

N.B. : avant d'installer un WES, vérifiez qu'il n'existe pas déjà ou désinstallez le précédent WES avant d'installer le nouveau.

Pour installer le WES automatiquement, dans la section WES Kyocera HyPAS cliquez sur le bouton Configurer.

Cette installation se déroule en plusieurs étapes listées dans la section Etat.



 

è Lorsque le WES est correctement installé, le message "L'application embarquée a été configurée avec succès" s'affiche.
