Modèles compatibles

La liste suivante référence des modèles testés avec Watchdoc. Les modèles qui n'y figurent pas n'ont jamais été testés. Pour savoir si un modèle non-listé est compatible, adressez votre demande via Connect.

La technologie Open Platform

  • v2.3 est requise pour l'authentification LDAP ;

  • v3.0 est requise pour l'authentification MDS et la mise en oeuvre des quotas.

Le navigateur embarqué (EWB) doit être activé.

 

Sur les modèles e-bridge N, la version de firmware supportée est la version 10.30 au minimum.

Lecteurs de badges

Les lecteurs de badge Elatec TWN3 et TWN4 sont supportés.

Les périphériques supportent d’autres modèles de lecteurs :

  • lecteurs Elatec en mode COM ;

  • lecteurs Cartadis TCM3.

Cependant, aucun support n'est assuré par Doxense pour ces lecteurs.

Une suggestion ? Une demande de correction ? Exprimez-vous !
