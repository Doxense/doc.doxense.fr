Watchdoc - Tarifs - Créer et configurer un tarif

Prérequis

Un tarif s'appuyant sur des unités monétaires, il est nécessaire d'en avoir configuré au moins une préalablement (cf. Watchdoc - Unités monétaires - Configurer) pour configurer un tarif.

Si les unités n'ont pas été configurées lors de l'installation initiale, cliquez sur Paramétrage monétaire pour accéder à l'interface de configuration.

Créer un tarif simple

Pour créer un tarif :

  1. dans la liste Tarifs & Coûts, cliquez sur le bouton Créer un nouveau tarif simple :

è Vous accédez à l'interface Création d'un tarif composée des sections Tarif et Coût à la page.

Configurer le tarif

Section Tarif

Complétez les zones suivantes :

  • Nom : attribuez un nom au tarif créé. 
    Ce nom s'affiche uniquement dans les interfaces d'administration et doit décrire le plus précisément possible le tarif. Par exemple : "Multifonction [Marque du constructeur]", "Contrat [Nom du fournisseur] 2026", "A4 monochrome sur MFP".
    Régulièrement, pour les tarifs servant à l'évaluation des coûts, c'est le nom du contrat "Coût à la page" liant l'organisme au fournisseur qui est attribué au tarif. Car c'est dans le contrat que les coûts réels d'impression englobant toutes les charges liées aux périphériques d'impression (matériel, maintenance et consommables) sont définis précisément.

  • Description: complétez le nom de votre tarif par toute information permettant de mieux le définir. Ces informations ne s'affichent que dans les interfaces d'administration.
    Si le nom du tarif est celui du contrat "Coût à la page", fournissez en Description les informations précisant ce contrat (nom du fourniseur, coordonnées, date de début et de fin de contrat, etc.) :


Section Coût à la page

Le coût à la page d'une impression peut varier en fonction du format du papier ou du mode d'impression (monochrome ou couleur). Précisez ces coûts en complétant la Table de prix :

  1. indiquez le coût à la page pour une impression monochrome au format A4 ;

  2. cochez la case A3 pour autoriser ce format d'impression et indiquez le coût à la page pour une impression monochrome ;

  3. cochez la case Couleur pour autoriser l'impression couleur et indiquez le coût à la page pour une impression couleur au format A4 et/ou A3 :

     

  4. Cliquez sur le bouton pourValider la configuration du tarif.


N.B. :

  • une page est considérée comme une impression couleur dès qu'un caractère ou un élément graphique coloré y est détecté ;

  • le coût d'un document correspond à la somme du coût de chaque page prise individuellement ;

  • un document recto-verso (duplex) n'est pas tarifé par feuille physique (1), mais par côté imprimé (2) ;

  • dans Watchdoc un format 'A4' est une page dont la surface est égale ou inférieure à 1,5 fois un A4, soit environ 935 cm2 (A4, A5, A6, Letter, Executive) ;

  • dans Watchdoc, un format 'A3' est une page dont la surface est supérieure à 1,5 fois un A4, soit environ 935 cm2 (A2, A1, Legal, Ledger, ...). Ce format est parfois appelé "A3+" ou "grand format".

Appliquer le tarif sur une file

  1. Rendez-vous dans la liste des files depuis le Menu principal > section Exploitation > Files d'impression, emplacements, groupes de files &pools ;

  2. dans la liste Files d'impression, cliquez sur la file sur laquelle vous souhaitez appliquer le tarif ;

  3. dans l'interface de gestion de la file, cliquez sur l'onglet Tarification) :

     

èles tarifs déjà appliqués s'affichent dans les sections Calcul du coût à la page ou Calcul du prix.

 

  1. en fonction de la politique tarifaire appliquée, vous pouvez configurer

    1. le Calcul du coût à la page : il s'agit du prix moyen par page (côté) imprimée (couleur/n&b, A3/A4) et tient compte du coût du papier, achat ou location machine et services (TCO).

      Cliquez sur Configurer le coût à la page et précisez :

      • Groupe : cochez ce bouton si le coût appliqué à la file doit être le même que le coût appliqué au groupe auquel la file appartient ;

      • Inconnu : cochez ce bouton si vous ne souhaitez pas évaluer le coût de l'impression sur cette file ;

      • Fixe : cochez ce bouton et sélectionnez un tarif dans la liste pour appliquer un coût à la page défini selon des critères précis ;

      • Variable : cochez ce bouton pour appliquer un tarif (prédéfini) en fonction de conditions configurées dans les filtres.

    • le Calcul du prix : il s'agit du coût réel majoré qui est facturé à l'utilisateur pour rentabiliser l'impression :

      Cliquez sur Configurer le calcul du prix et précisez :

      • Groupe : cochez ce bouton si le prix appliqué à la file doit être le même que le prix appliqué au groupe auquel la file appartient ;

      • Inconnu : cochez ce bouton si vous ne souhaitez pas évaluer le prix de l'impression sur cette file ;

      • Fixe : cochez ce bouton et sélectionnez un tarif dans la liste pour appliquer un prix défini selon des critères précis ;

      • Variable : cochez ce bouton pour appliquer un tarif (prédéfini) en fonction de conditions configurées dans les filtres.

         

     

    N.B. : si la fonction Quota est activée, précisez le(s) quota(s) utilisé(s) pour le paiement dans la section Paiement automatique.
