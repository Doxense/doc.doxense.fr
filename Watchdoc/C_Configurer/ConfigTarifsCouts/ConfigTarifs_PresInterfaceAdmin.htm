Watchdoc - Tarifs et Coûts

Présentation de l'interface d'administration des Tarifs et coûts

Accéder à l'interface de configuration

  1. Depuis le Menu principal de l'interface d'administration, dans la section Configuration, cliquez sur Tarifs & Coûts.

→ Vous accédez à l'interface Tarifs & Coûts.

Description des outils de la liste

La liste affiche l'ensemble des tarifs déclarés dans l'application Watchdoc®.

 

Chaque ligne présentant un tarif comporte les outils suivants :

  • Le bouton Modifier ce tarif permet d'ouvrir le tarif dans une interface spécifique afin d'en étudier la configuration et de la modifier ;

  • le bouton Dupliquer ce tarif permet de prendre un tarif comme modèle pour en créer un nouveau plutôt que de le reconfigurer ex-nihilo.

Avant d'utiliser l'outil de duplication, vérifiez qu'il n'y aura pas plus de paramètres à modifier qu'à conserver.

  • le bouton Supprimer ce tarif permet de supprimer tarif qui n'est plus utilisé. La suppression n'est effective qu'après confirmation.

  • le bouton Créer un tarif simple situé sous la liste ouvre l'interface de création d'un tarif.

Informations de la liste Tarifs simples

Chaque ligne affiche les informations constituant le tarif :

  • le Nom et la description du tarif ;

  • le Format : la monochromie ou la couleur ainsi que le format des impressions auquel s'applique le tarif ;

  • Prix : les différents coûts définis dans le tarif (en fonction des couleurs et formats) ;

  • Ratio A4 : le ratio de comparaison des coûts par rapport à une page A4 monochrome ;

  • Prix pour 500 feuilles : coût indicatif de l'impression d'une ramette ;

  • Ratio global : ratio permettant une vérification rapide de la cohérence du tarif. 

Quand on coche la case A3, par défaut le tarif proposé pour l'A3 est le double du tarif d'une page A4, mais il peut être modifié.

Informations de la liste Tarifs complexes

Chaque ligne affiche les informations constituant le tarif :

  • le Nom et la description du tarif ;

  • le Prix de base : la monochromie ou la couleur ainsi que le format des impressions auquel s'applique le tarif ;

  • Opé. : affiche les opérations appliquées, telles que le coefficient d'ajustement ou la multiplication par un ratio de surface.

  • Tranche : affiche une information relative à la variabilité du tarif par tranches de pages imprimées.
