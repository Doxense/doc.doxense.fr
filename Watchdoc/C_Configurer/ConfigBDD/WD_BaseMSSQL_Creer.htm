Watchdoc - Base de données MS SQL - Créer une base de données

Prérequis

Avant d'installer la base de données MS SQL, vérifiez les prérequis suivants :

  • la version de MS SQL est compatible avec la version de Watchdoc (cf. les prérequis) ;

  • l'outil MS SQL Visual Studio Management Service est installé sur le serveur (librement téléchargeable depuis le site Microsoft).

     

La base de données MS SQL peut être installée sur le même serveur que Watchdoc (mode autonome) ou sur un autre serveur (mode classique et mode déporté).

Dans le cas où le serveur SQL est distant, vérifiez le prérequis suivants :

  • TCP/IP est activé dans SQL Server Configuration Manager ;

  • le service SQL Browser est démarré afin que Watchdoc puisse afficher la liste des serveurs et les instances disponibles ;

  • le port TCP 1433 est ouvert sur le pare-feu pour l'instance ;

  • le port UDP 1434 est ouvert sur le pare-feu pour SLQ Server Browser.

Procédure

Pour créer la base de données :

  1. lancez l'outil Microsoft® SQL Server Management Studio ;



  2. depuis SQL Server Management, connectez-vous au serveur SQL de Watchdoc (si la base n'est pas locale) ;

  3. sélectionnez l'entrée Bases de données, puis cliquez droit pour accéder au menu ;

  4. sélectionnez Nouvelle base de données...

  1. dans l'interface Nouvelle Base de données, saisissez le Nom de la base de données dans la zone dédiée ;



  2. puis cliquez sur le menu Options et choisissez le Mode de récupération Simple :



  1. cliquez sur OK pour valider la création de la base de données.

  2. Réitérez l'opération pour ajouter la base watchdocpmv si vous souhaitez utiliser la fonction Quotas de Watchdoc.

è Une fois la base de données créée, vous pourrez la connecter à Watchdoc lors de l'installation initiale (cf. Configuration initiale - Connecter la base de données).
