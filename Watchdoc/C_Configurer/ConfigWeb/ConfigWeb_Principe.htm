Watchdoc - Configurer un profil de déblocage web (page "mon compte")

Principe

Qu'est-ce qu'un profil de déblocage web ?

Watchdoc dispose d'un site web (appelée la page "Mon compte") destiné aux utilisateurs afin de permettre à ceux-ci d'accéder à différentes fonctions de l'outil, selon la manière dont ce site web est paramétré.

 

Depuis cette interface, l'utilisateur accède :

  • à la liste de ses travaux qu'il peut gérer débloquer ;

  • à ses statistiques d'impression ;

  • à ses comptes et opérations (si les quotas sont activés par l'administrateur) ;

  • à ses archives (si l'archivage est activé par l'administrateur).



L'interface web comportant plusieurs fonctions, il est possible de configurer des profils répondant à des besoins distincts :

  • consulter ses impressions ;

  • consulter ses comptes ;

  • débloquer ses impressions ;

  • etc.

Le site permet notamment à l'utilisateur de débloquer ses impressions dans le cas où le périphérique ne dispose pas de WES WES (Watchdoc Embedded Solution - Solution Embarquée Watchdoc) WES est le nom donné à l'interface Watchdoc embarquée sur les périphériques d'impression. Il existe des interfaces spécifiques à chaque périphérique tiers et donc autant de WES que de constructeurs de périphériques. Ces interfaces servent à gérer l'impression depuis le périphérique lui-même., mais peut fonctionner conjointement au WES.

Principe de la configuration

Différents sites web peuvent être conçus en fonction de besoins différents. Par exemple, un profil web de déblocage en français pour les utilisateurs francophones et un autre en anglais pour les utilisateurs anglophones :

 

 

Le profil web configuré est mis à disposition des utilisateurs sous forme d'une adresse URL communiquée par les administrateurs de Watchdoc (par exemple http://print1/watchdoc/default.asp?).

 

Pour informer vos utilisateurs du profil de déblocage web qu'il doit utiliser,

  1. configurez vos profils web ;

  2. dans la liste des profils, cliquez sur la loupe permettant de prévisualiser l'un de ces profils ;

     

  3. enregistrez l'url qui s'affiche dans le navigateur ;

  4. communiquez cette url à vos utilisateurs.
