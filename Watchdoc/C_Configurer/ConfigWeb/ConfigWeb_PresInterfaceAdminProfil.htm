Informations de la liste des profils de déblocage web (page "Mon compte")

La liste Profils de configuration des instances de déblocage web et mobile affiche l'ensemble des interfaces déclarées dans l'application Watchdoc :

 

Les interfaces affichées dans ces listes sont accompagnées des informations suivantes :

  • la colonne Identifiant affiche l'identifiant interne de l'interface web, affiché dans les interfaces d'administration uniquement ;

  • la colonne Nom affiche le nom, plus explicite que l'identifiant, affiché dans les interfaces d'administration uniquement ;

  • la colonne Titre affiche le nom de l'interface telle qu'il est affiché aux utilisateurs, accompagné d'une icone précisant son mode d'adaptation linguistique ;

    • (ou drapeau d'un autre pays) indique la langue dans laquelle les libellés de l'interface sont rédigés ;

    • indique que l'interface s'affiche dans la langue définie dans le système d'exploitation du poste de l'utilisateur ;

    • indique que l'interface s'affiche dans la langue définie par l'administrateur lors de l'installation de Watchdoc.

  • la colonne Identification précise le mode d'authentification utilisé dans l'interface, accompagné d'une icone :

    • indique que l'utilisateur s'authentifie en saisissant ses compte et mot de passe dans un formulaire ;

    • indique que l'utilisateur s'authentifie à l'aide de son badge.

  • la colonne Paiement.indique le mode de paiement appliqué dans l'interface ;

    • gratuit : indique que l'impression n'est pas soumise à paiement ;

    • card indique que l'utilisateur paie à l'aide d'une carte approvisionnée ;

    • PMV : indique que l'utilisateur paie à l'aide d'un Porte-Monnaie Virtuel, c'est-à-dire d'un compte Watchdoc propre à l'utilisateur, approvisionné spécifiquement pour ses impressions ;

    • externe : indique que l'utilisateur doit payer manuellement un opérateur chargé de débloquer ses impressions.

  • la colonne Délais : indique, en minutes et secondes, les délais laissés à l'utilisateur avant rafraîchissement, déconnexion et redirection automatiques.

Outils de la liste

Chaque ligne présentant un profil web comporte les outils suivants :

  • le bouton Tester permet d'accéder à l'interface, telle qu'elle est configurée et qu'elle s'affiche aux utilisateurs ;

  • le bouton Définir comme profil par défaut permet d'appliquer au profil de configuration le statut spécifique de profil exploité en priorité par Watchdoc. Dans la liste, le profil par défaut est surligné et précédé de l'icone .

  • le bouton Editer permet d'ouvrir le profil dans une interface de configuration ;

  • le bouton Dupliquer ce profil permet de prendre un profil comme modèle pour créer un nouveau profil plutôt que de le configurer ex-nihilo.

    Avant d'utiliser l'outil de duplication, vérifiez qu'il n'y aura pas plus de paramètres à modifier qu'à conserver.

  • le bouton Supprimer ce profil est utilisé dans le cas où le profil n'est plus utile.
