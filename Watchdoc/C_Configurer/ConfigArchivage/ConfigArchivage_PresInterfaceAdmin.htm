Archivage - Présentation

Principes relatifs à l'archivage

Dans Watchdoc, la fonction d'archivage permet de conserver les fichiers de spool (.SPL et .SHD) sur le serveur. Elle répond à deux besoins précis :

  • pour l'utilisateur, le besoin de réimprimer rapidement un travail qui a été fermé sur le poste de travail ;

  • pour l'administrateur, le besoin de capturer les spools de travaux, de sorte de pouvoir les analyser lorsqu'un problème d'impression aléatoire et difficilement reproductible survient.

 

L'archivage peut être :

  • global, lorsqu'il s'applique automatiquement sans distinction à tous les travaux, d'impression ;

  • ciblé, lorsqu'il s'applique spécifiquement à des travaux d'impression qui ont été distinguésau moyen d'une règle (appliquée sur un groupe de file ou sur une file).

Prérequis

La fonction d'archivage occupant de la place sur l'espace disque, elle est désactivée par défaut sur Watchdoc®. Avant de l'activer, il convient de vérifier que l'espace disque est suffisant pour conserver les Archives.

Etapes

Pour être activée, la fonction d'archivage doit être paramétrée au niveau :

  1. du serveur (comme expliqué dans ce chapitre),

  2. du groupe de files ou d'une file spécifique (cf. le chapitre relatif à la configuration d'une file ou d'un groupe de files).

Présentation de l'interface d'administration de l'archivage

Accéder à l'interface de configuration

Depuis le Menu principal de l'interface d'administration,

  1. dans la section Configuration, cliquez sur Configuration avancée.

  2. dans l'interface [SERVEUR_IMPRESSION] > Configuration avancée,, cliquez sur Archivage des documents :

     

→ Vous accédez à l'interface Configuration de l'archivage.

Présentation de l'interface

L'interface Configuration de l'archivage comporte 3 sections :

  • Paramètres : regroupe les informations propres à la fonction d'archivage ;

    • Répertoire : indiquez dans ce champ le chemin d'accès au répertoire de stockage des documents archivés ;

    • Compression : cochez la case si vous souhaitez comprimer les documents archivés et optez pour le format de compression souhaité (GZip et BZip2) ;

  • Mode Expert : les paramètres de cette section permettent de mettre en place un système d'audit des opérations d'archivage afin de vérifier qu'elles fonctionnent correctement :

    • cochez la case Surveiller les opérations d'archivage si vous souhaitez garder trace des opération d'archivage dans un fichier qui sera sauvegardé ;

    • cochez la case Enregistrer les requêtes d'indexation dans le journal de l'application si vous conserver les requêtes d'indexation de l'archivage.
