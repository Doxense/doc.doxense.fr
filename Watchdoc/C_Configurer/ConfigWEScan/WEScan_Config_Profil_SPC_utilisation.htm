WEScan - ScanProfilesCustomizer - Présentation

Principe

Un

profil (de scan / de numérisation) est une somme de paramètres (format, couleur, orientation, type du document, etc.) définissant la manière dont un document est numérisé.
Enregistré dans un fichier JSON, le profil de numérisation peut être modifié à l'aide de l'exécutable ScanProfilesCustomizer.

 

N.B. : dans un domaine, pour garantir la disponibilité des profils de numérisation sur tous les serveurs qui dépendent du serveur maître en cas de panne de la base de données interserveur, il convient de dupliquer tous les fichiers JSON des profils de numérisation du serveur master sur les serveurs qui en dépendent.
Pour cela, copiez le dossier de profils de numérisation personnalisés (dans lequel vous avez enregistré vos profils personnalisés) de votre serveur master sur le(s) serveur(s) qui en dépendent.

Procédure

Accéder à ScanProfilesCustomizer

  1. Accédez en tant qu'administrateur au serveur sur lequel est installé Watchdoc ;

  2. à l'aide d'un explorateur, rendez-vous dans le dossier d'installation de Watchdoc (par défaut : C:\Program Files\Doxense\Watchdoc) ;

  3. cliquez sur lScanProfilesCustomizer pour ouvrir l'outil :

 

Modifier un profil de scan

  1. Dans l'outil ScanProfilesCustomizer, section Json File, cliquez sur le bouton Browse pour parcourir la liste des marques de périphériques compatibles avec WEScan ;

  2. sélectionnez le fichier JSON correspondant à la marque du profil à modifier ;

  3. cliquez sur Open ;

  4. dans la section Profiles s'affichent tous les profils déclarés pour la marque sélectionnée ;

  5. sélectionnez un profil : dans la liste Settings s'affichent tous les paramètres définissant le profil ;

  6. les valeurs de chaque paramètre s'affichent à droite : modifiez-les selon vos besoins :

     

  7. Une fois les paramètres définis, cliquez sur Save pour les sauvegarder.

Créer un profil de scan

  1. Dans l'outil ScanProfilesCustomizer, section Json File, cliquez sur le bouton Browse pour parcourir la liste des marques de périphériques compatibles avec WEScan ;

  2. sélectionnez le fichier JSON correspondant à la marque pour laquelle vous souhaitez créer un profil ;

    cliquez sur Open ;

  3. dans la section Profiles s'affichent tous les profils déclarés pour la marque sélectionnée ;

  4. cliquez sur le bouton + :

  5. dans la fenêtre Create Profile, saisissez :

    • Id: l'identifiant du profil ;

    • Label : un libellé caractérisant le profil

  6. cliquez sur Create :

     

  7. l'identifiant du profil créé apparaît dans la liste, préfixé par un identifiant de marque. Cliquez dessus pour le sélectionner ;

  8. tous les paramètres définissant le profil s'affichent dans la section Settings ;

  9. modifiez-les selon vos besoins ;

  10. Une fois les paramètres définis, cliquez sur Save pour les sauvegarder.
