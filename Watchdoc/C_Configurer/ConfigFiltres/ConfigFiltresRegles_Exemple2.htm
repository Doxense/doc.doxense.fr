Activer la suggestion sur la file A

La règle vise à proposer à l'utilisateur de modifier ses choix d'impression initiaux en optant pour une impression plus responsable : recto-verso plutôt que recto et noir&blanc plutôt que couleur.
Ces critères ne sont pas imposés mais juste suggérés.
Cette politique ne peut s'appliquer que sur un WES supportant la transformation de spools.

 Aucun rôle = la suggestion s'applique à tous les utilisateurs

Rediriger les travaux d'impression volumineux

La règle stipule que les documents dépassant 500 pages ne peuvent être imprimés sur la file A.
Ces travaux sont donc redirigés sur la file B optimisée pour les impressions volumineuses.

 Aucun rôle = la règle s'applique à tous les utilisateurs
