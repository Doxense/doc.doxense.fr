Watchdoc - Filtres et règles - Présentation de l'interface d'administration des filtres

Accéder à l'interface de configuration

Depuis le Menu principal de l'interface d'administration,

  1. dans la section Configuration, cliquez sur Filtres:

è Vous accédez à la liste des filtres déclarés et prédéfinis par défaut dans Watchdoc.

Description de la liste

  • la section Filtres affiche la liste des filtres configurés spécifiquement pour vos besoins. Le bouton permet d'afficher les règles appliquées dans chaque filtre ;

  • la section Filtres prédéfinis affiche la liste des conditions prédéfinies et non modifiables dont vous disposez pour concevoir vos filtres.

Ces listes affichent les informations suivantes :

  • le Nom du filtre ;

  • dans la colonne Message s'affiche l’information notifiée à l'utilisateur lorsque le filtre est déclenché ;

  • des informations sur l'objet du filtre :

    • des types d'utilisateurs ;

    • des périphériques d'impression ;

    • des types de documents ;

    • des dates et durées.

 

Chaque ligne présentant un filtre comporte les outils suivants :

  • Le bouton Modifier ce filtre permet d'ouvrir le filtre dans une interface spécifique afin d'en étudier la configuration et de la modifier ;

  • le bouton Dupliquer ce filtre permet de prendre un filtre comme modèle pour en créer un nouveau plutôt que de le configurer ex-nihilo.
    Avant d'utiliser l'outil de duplication, vérifiez qu'il n'y aura pas plus de paramètres à modifier qu'à conserver.

  • le bouton Supprimer ce filtre permet de supprimer un filtre qui n'est plus utilisé. La suppression n'est effective qu'après confirmation.
