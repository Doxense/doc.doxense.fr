Watchdoc - Filtres et règles - Politique d'impression - Exemples d'application

Exemples

Interdire l'impression en couleur sur une file

La règle suivante stipule que les documents en couleur ne sont pas autorisés sur la file et qu'ils sont donc supprimés.

 

Restreindre l'impression en couleur sur une file

La règle suivante stipule que :

  • les documents en couleur ne peuvent être imprimés sur la file que par les membres du service Marketing ;

  • les travaux d'impression en couleur demandés par les membres d'autres services sont supprimés.

 

Soumettre l'impression en couleur à validation

La règle suivante stipule que

  • que les documents en couleur peuvent être imprimés sur la file par les membres du service Marketing ;

  • et que les travaux d'impression en couleur demandés par les membres d'autres services doivent être validés par un supérieur.

    Ces travaux sont donc bloqués sur la file avant impression ou suppression.

 

Rediriger les travaux d'impression volumineux

La règle suivante stipule que les documents dépassant un certain nombre de pages ne peuvent être imprimés sur la file.
Ces travaux sont donc redirigés sur une file optimisée pour les impressions volumineuses.
