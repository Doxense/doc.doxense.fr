Watchdoc - Service - Configurer

Configurer les unités monétaires

Dans cette section, vous configurez les informations relatives aux coûts et refacturation des impressions.

  • Global : cochez la case dans le cas où vous configurez les unités monétaires sur le serveur maître et que vous souhaitez que ce paramètre soit répliqué sur tous les autres serveurs du domaine. 

Section Coût à la page

La section Coût n'est utilisée qu'à des fins statistiques et suivi et de contrôle des coûts, contrairement à la section Refacturation dans laquelle sont définis les coûts effectivement facturés à l'utilisateur.

Le coût à la page est défini après calcul des matériels, logiciels et consommables entrant dans la fonction d'impression (coût, à l'achat et en maintenance, des périphériques d'impression, coût des ramettes, des cartouches, de l'électricité, etc.).

 

  • Profil : choisissez le format dans lequel s'affichent les coûts de fonctionnement de l'impression (coût des consommables, coût de l'impression à la page, par exemple). Vous avez le choix entre plusieurs formats prédéfinis ou la possibilité de paramétrer votre propre format. Ce format d'affichage est appliqué dans les interfaces Watchdoc informant sur les coûts (notamment statistiques et rapports).

  • Formule : si vous avez sélectionné le choix Paramétrage personnalisé, indiquez le nombre de décimales ainsi que la devise à afficher. Le format que vous définissez est appliqué en exemple au nombre Pi.

Section Refacturation

  • Profils : choisissez le format dans lequel s'affichent les coûts de paiement ou de facturation appliqués à l'impression. Vous avez le choix entre plusieurs formats prédéfinis ou la possibilité de paramétrer votre propre format. Ce format d'affichage est appliqué dans les interfaces Watchdoc informant sur le tarif d'impression (notamment dans les interfaces utilisateur).
    Optez pour le choix 123 - Pas d'unités si l'impression est gratuite.

  • Formule :si vous avez sélectionné le choix Paramétrage personnalisé, indiquez un coefficient, le nombre de décimales ainsi que la devise à afficher. Le format que vous définissez est appliqué en exemple au nombre Pi.
