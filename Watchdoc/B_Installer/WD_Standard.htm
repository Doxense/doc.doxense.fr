Watchdoc Standard

Présentation

En fonction du nombre de serveurs mis à disposition des services nécessaires à son fonctionnement, Watchdoc Standard peut être installé selon différents modes :

  • mode autonome : les composants kernel, base de données et serveur web IIS sont installés sur le même serveur ;

  • mode classique : les composants kernel et serveur web IIS sont installés sur le même serveur, tandis que la base de données est installée sur un serveur dédié ;

  • mode déporté : les composants kernel et service d'impression sont installés sur le même serveur, tandis que le serveur web IIS et la base de données disposent chacun d'un serveur dédié.

    Pour des raisons de sécurité, Doxense déconseille l'installation en mode déporté.


    Cliquez sur l'image pour l'agrandir


N.B. : dans les 3 cas, le contrôleur de domaine est installé sur un serveur dédié.

 

En fonction du mode choisi, la procédure d'installation varie.

Pour les modes autonome et classique, l'installation se déroule en 2 étapes :

  1. installation du kernel et du site web à l'aide de l'assistant de configuration ;

  2. configuration de Watchdoc.

Pour le mode déporté l'installation se déroule en 3 étapes :

  1. installation du kernel à l'aide de l'assistant de configuration ;

  2. installation du site web à l'aide de l'assistant de configuration ;

  3. configuration de Watchdoc.
