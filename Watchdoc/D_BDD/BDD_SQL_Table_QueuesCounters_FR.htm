Table queuescounters

Description

La table queuescounters contient toutes les informations collectées en SNMP sur les matériels d’impression. Cette table inscrit N (nombre de file d’impression) lignes toutes les 5 minutes et, chaque heure, ces X lignes sont consolidées en une seule pour chaque file d’impression.
