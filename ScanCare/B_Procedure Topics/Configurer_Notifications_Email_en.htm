Configuring E-mails® notifications

Watchdoc ScanCare® has a system of e-mail notifications to alert administrators of any operating issues that may arise

Once the installation is complete, the program prompts you to setup this e-mail notification function:

  1. From the ScanCare setup interface, click on Tools > Options,

  2. From the Setting interface, click on the Alerts tab,

  3. From the Email preferences section, tick the Activate Email alerts box

  • Server: From the list, choose the name of the message server (SMTP, by default);

  • Port :

  • By default: By default port 25 is used. To use another port, untick the box and enter the port number in the field provided,

  • SSL Management Information Base (en française, "Base d'information pour la gestion du résesau") est un ensemble structuré d'informations relatif à une entité d'un réseau (imprimante ou périphérique multifonction, par exemple). Ces informations peuvent être récupérées ou parfois modifiées par un protocole tel que SNMP.:Tick the box if the server is secured using this protocol.

  • SMTP authentication: if access to the SMTP server is secured, enter:

  • User: The name of the user allowed to access the server;

  • Password: The password of the user allowed to access the server.

  • User information:

  • Sender name: Enter the name of the notification sender. If there are a number of possible senders, you can enter a number of names and they will be displayed in the list,

  • Sender e-mail: Enter the sender's e-mail address. If there are a number of possible senders, you can enter a number of e-mail addresses and they will be displayed in the list.

  • E-mail recipient: Enter the e-mail address of the person to be informed if there is a malfunction (e.g. the person in charge of printing or the IT services department).

  • Prefer logged on users credentials: Tick this box to notify users who are logged on to the devices. In this case the message can also be sent in copy to the administrator using the CC setting that follows.

  • CC: Tick this box to send a copy of the e-mail to the (one or more) administrator(s) entered in the Sender name field;

  • Default message body: Used these fields to enter the content (subject header and text) for the message sent to the recipient in case of malfunction.

  1. Click on OK to validate the settings defined for Email alerts.

You can fill-in the fields using the functions and variables with a dedicated tool. To access it, click on the logo that matches the field to configure and follow the procedure described in the chapter called Using the functions and variables.
