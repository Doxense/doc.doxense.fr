Wathdoc ScanCare - Configurer

Configurer les alertes e-mails

Watchdoc ScanCare dispose d'un système de notifications par e-mail pour alerter les administrateurs d'éventuels problèmes de fonctionnement.

Au terme de l'installation, le programme vous invite à paramétrer cette fonction d'alerte mail :

  1. dans l'interface d'administration ScanCare, cliquez dans le menu Outils > Options ;

  2. dans l'interface Paramètres généraux, cliquez sur l'onglet Alertes ;

  3. dans la la section Préférences Email, cochez la case Activer les alertes Email ;

    • Serveur : dans la liste , sélectionnez le nom du serveur de messagerie (SMTP, par défaut) ;

    • Port :

    • Par défaut : par défaut, le port utilisé est le 25. Si vous souhaitez utiliser un autre port, décochez la case et saisissez le numéro du port dans le champ dédié ;

    • SSL Management Information Base (en française, "Base d'information pour la gestion du résesau") est un ensemble structuré d'informations relatif à une entité d'un réseau (imprimante ou périphérique multifonction, par exemple). Ces informations peuvent être récupérées ou parfois modifiées par un protocole tel que SNMP.: cochez la case si le serveur est sécurisé au moyen de ce protocole ;

    • Authentification SMTP : si l'accès au serveur SMTP est sécurisé, saisissez :

    • Utilisateur : le nom de l'utilisateur habilité à accéder au serveur ;

    • Mot de passe : le mot de passe de l'utilisateur habilité à accéder au serveur.

    • Informations utilisateur :

    • Nom de l'expéditeur : saisissez le nom de l'expéditeur des notifications. S'il existe plusieurs expéditeurs possibles, vous pouvez saisir plusieurs noms qui s'affichent alors dans la liste ;

    • Email de l'expéditeur : saisissez l'adresse mail de l'expéditeur.  S'il existe plusieurs expéditeurs possibles, vous pouvez saisir plusieurs adresses mail qui s'affichent alors dans la liste.

    • Email du destinataire : saisissez l'adresse mail de la personne devant être prévenue en cas de dysfonctionnement (par exemple, le responsable de l'impression ou le D.S.I).

    • Préférer les utilisateurs identifiés et référencés : cochez cette case pour notifier les utilisateurs authentifiés sur les périphériques. Dans ce cas, le message peut aussi être envoyé en copie à l'administrateur grâce au paramètre CC qui suit.

    • CC : cochez cette case pour envoyer un mail en copie à l'administrateur (ou les administrateurs) saisi dans le champ Nom de l'expéditeur.

    • Sujet et corps de texte par défaut : saisissez dans ces champs le contenu (objet et texte) du message envoyé au destinataire en cas de dysfonctionnement.

       

  1. Cliquez sur OK pour valider les paramètres définis pour les alertes par Email.

     

Vous pouvez remplir les champs avec des fonctions et variables à l'aide d'un outil dédié. 
Pour y accéder, cliquez sur le logo "Variables" correspondant au champ à paramétrer et suivez la procédure décrite dans le chapitre Watchdoc ScanCare - Utiliser les fonctions et les variables.
