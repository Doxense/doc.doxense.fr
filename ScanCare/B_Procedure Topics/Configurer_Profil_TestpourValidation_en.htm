Watchdoc ScanCare - Configuring a scanning profile - Procedure

Configuring a profile test for validation

The following procedure describes how to quickly setup a profile and assign it to a device to quickly validate the proper operation of Watchdoc ScanCare. It does not describe all of the setup options offered by the solution.

The profile that is configured to test the solution should allow scanning documents, cleaning them (Despeckle) and straightening them (Deskew) before sending to a preset e-mail address (e.g. yours).

Creating a profile

  1. From the Watchdoc ScanCare setup interface, click on the Add button located above the list of profiles or at the bottom of the main window:

 

  1. From the Profiles dialogue box that is displayed, choose the Embedded Device type and configure it:

    • Name: Enter the name assigned the profile: "ScanToMail", for example. As this name is displayed on the device's display, the number of characters is limited,

    • Description:If necessary, complete the name by entering a summary description: "Scan and send the document by mail", for example. As this description is displayed on the device's display, the number of characters is limited,

    • ProfileType : From the list, choose the type of profile:

      • scanner : allows to scan the submitted document after first applying various kinds of processing to it.

  2. Click on the OK button to create the profile.

 

→The profile that is created is displayed on the left in the profiles column.

Configuring the profile

Click on the profile displayed in the left hand column to access its setup interface. The window that is opened by default is the one for setting up the index fields.

Setting up an index field

  1. From the right hand Index field, click on the Add button.

  2. From the Add new field window, enter the "Title" in the Display field.

  3. Retain the Text type.

  4. In the Usage section, select Default.

  5. Tick the Input is mandatory box.

  6. Click on the OK button to save the new field :

 

Configuring the processing

Configure the one or more processing functions to apply to the profile:

  1. click on the Processing button;

  2. Tick the Iimage Processing box and click on the Preferences button ;

 

  1. In the Preferences>Image processing box, tick the Despeckle and Deskew boxes.

  2. Click on the OK button to save the processing applied to the scanned documents.

  3. Click on the Test the settingsbutton to confirm the processing.

 

Configuring the output

  1. Configure the format that the scanned documents will be converted to, namely an e-mail message sent to a preset address:

    1. Click on the Output button;

    2. From the Output types menu offered, click on the Email section, then on the Email button;

    3. Tick the Enabled box,

    4. From the E-mail Server section, click on the Edit button to access the setting interface,

If you already setup the e-mail server when setting up the notifications (see Configuring a test profile for validation), then these settings are reused by default in the e-mail output settings interface. If the SMTP Server is the same, retain these settings

  1. From the E-Mail Server>Server Type settings interface, keep the SMTP Server box ticked by default;

  2. Configure the E-Mail Settings section:

  3. Enter the e-mail Sender name (or sender department);

  4. Enter the Sender e-mail address (or sender department);

  5. Enter the SMTP Server adress;Enter the Port number (if it is different from the default port) and thick the box if the port is secured by SSL protocol;

  6. Tick the Prefer logged on users credentials box to use the user's connection credentials to authenticate them with the SMTP server

  7. Tick the SMTP Authentication box and fill-in the following fields to designate the user allowed to access the SMTP server:

  8. Enter the User name allowed to connect to the SMTP server

  9. Enter the Password for the user allowed to connect to the SMTP server.

  10. Click on the OK button to save the e-mail server settings.

  2. From the list of E-mail Distribution addresses, click on the Add button, ;

  3. From the recipient's E-mail Distribution box, fill-in the following fields:

  4. Name: Enter the name of the recipient (or group of recipients) for the e-mail ;

  5. Email address: Enter the e-mail address of the preset recipient (or group of recipients) for the e-mail message;

  6. Opt for A, CC ou Bcc depending on the role played by the recipient;

If the recipient's name and e-mail address are not preset values, then you have a function and variables manager to fill-in these fields using items from the system.

  2. Click on the Save button to save your e-mail output.

Assigning the profile to a device

Once the profile is created, you need to make it available to users on one (or more) of the device(s) in your installation.

  1. Click on the Options button.

  2. From the Device Selection section, click on the Edit button;

  3. From the Devices List, select the one or more devices you wish to assign the profile to;

  4. Click on the OK button to save the profile assigned to the device.

 

Testing the Watchdoc ScanCare service.

Once the profile is assigned to a device, you can test the Watchdoc ScanCare service from the device

Login to the device,

  1. From the screen on this device, select the ScanCare service (the interface seen by the user will vary with the brand of the device),

  2. From the form displayed on the screen on the device, enter the title of the document you wish to scan and send by e-mail,

  3. Start scanning your document;

  4. Receive the document in the preset e-mail mailbox;

  5. If the profile is correctly setup, you will receive the scanned document by e-mail.
