Watchdoc ScanCare - Compatibilities Supported print devices

Watchdoc ScanCare is compatible with printers and scanners from the following manufacturers:

Canon® ScanFront® (scanners)

  • ScanFront® 220/220P

  • ScanFront® 300/ 300P

  • ScanFront® 330 (à partir de la version Watchdoc ScanCare 7.0)

  • ScanFront® 400 (à partir de la version Watchdoc ScanCare 7.0.740).

Fujitsu® (scanners)

  • fi-6010

  • ScanSnap N1800

  • N7100

 

N.B.: On the N7100, the built-in keyboard generates positioning errors. We therefore recommend using a USB keyboard for optimum Watchdoc ScanCare performance..

Hewlett Packard®

HP® devices are supported starting with Watchdoc ScanCare version 8.4

 

  • Watchdoc ScanCare supports HP MFPs which support OXPd.NET (Open Extensibility Platform device layer) enabled devices and provide the WebKit based browser.

  • (Indication that your device does provide those features is OXPd JavaScript support and a FurureSmart family firmware.

  • Further requirement is a control panel that sports a resolution of either 800 x 600, 1024 x 600 (will use 800x600) or 1024 x 800 pixels.

  • The current internal HP display series that fulfill those requirements are Oriani, Phantom, Pharos, PhotonR and Pulsar. (For all those control panels the versions with and without Pull-Out keyboard are supported).

 

AccordingHP information, only >= 8"panels are currently supporting the required resolutions.

 

Installation method

For HP devices automatic installation via device management is supported.

 

Network Communication

The following ports are used by default for the communication between Watchdoc ScanCare and the device

  • TCP 80, 9000 (http)

  • TCP 443, 4443 (https)

  • TCP 7627 (OXPd)

  • UDP 161 (SNMP)

  • TCP 9100 (direct print)

Konica Minolta®

All Konica Minolta® devices with a web browser are supported, but

Devices A3 device of the series xx7 or xx8 must include the web browser (activated from the device's administration page).

Devices that do not have a web browser must have the following components:

  • A LK-101v3 web browser license kit ;

  • Memory : UX-204.

Kyocera®

All compatibles HyPas (A3 et A4) devices are supported.

The following overview of the supported Lexmark® devices is subject to change and exemplary.To determine the actual supported eSF version for a certain device, follow the reference provided by Lexmark.

Oki

All MC700 and ES94/91xx series devices and devices using the sXP platform are supported.

 

All follorwing devices are supported:

  • MC873 / MC853, ES8473 / ES8453 (A3 color MFP series of sXP2)

  • MB562 / MB492, ES5162 / ES4192 (A4 mono MFP series of sXP2).

 

To upgrade to the latest firmware for the devices listed above, use the following link:
http://global.okiprintingsolutions.com/FWInterface.nsf/frmSelectLanguage

N.B.: For OKI Din A3 multi-function systems, documents can only be scanned in landscape format.

Ricoh

Ricoh® devices are supported starting with ScanCare Version 8.2.

 

Watchdoc® ScanCare supports all SOP-enabled Ricoh® devices.
If a device does not feature a Ricoh® Smart Operation Panel, please check if there is an optional upgrade available for your model, so the Ricoh SmartSDK, successor of SDK Type-J (SDK/J), the Web API and the WebBrowserNX can be made available for the seamless integration of Watchdoc® ScanCare.

Samsung

All of the devices based on XOA with a 7, 8.9 and 10.1 inches display are supported.
Watchdoc ScanCare Samsung® supports all XOA-based Samsung® MFPs with a display size of 7", 8,9" and 10,1".

 

Doxense hereby announces, effective 1st March 2021 Watchdoc ScanCare Samsung edition is no longer available for purchase.
No further enhancements or support will be available for Watchdoc ScanCare Samsung Edition. This product will no longer be maintained after the current maintenance and support term ends for customers / end customers.

Sharp

All devices that are OSA 2 to 5 compatible systems are supported, except for Jupiter series (AR-M351N/U, AR-M451 N/U) and Dragon (AR-M550, AR-M620 and AR-M700).

 

N.B.: The AMX2 key must be activated on the OSA device.

Toshiba

The e-BRIDGE 3 model and later versions.

 

N.B.: The Toshiba Embedded Web Browser (EWB) must be activated on the peripheral (code GS-1020 on the External Interface Enabler). As this option is not available everywhere, check for the presence of this option with your Toshiba® contact.

Xerox

All EIP® enabled Xerox® devices including ColorQube™, VersaLink® and AltaLink®.
