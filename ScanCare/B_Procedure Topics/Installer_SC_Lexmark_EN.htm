Install and configure Watchdoc ScanCare manually on a Lexmark device

Principle

Sometimes the automatic installation of Watchdoc ScanCare on a Lexmark device fails. In this case, a manual installation is required.

Procedure

Preamble

Before installing Watchdoc ScanCare manually, make sure that the Use profiles function is set to the No security parameter:

  1. Access the Lexmark device administration web interface (by entering the device's IP address in a web browser);

  2. On the Lexmark Device Administration page, click the Settings menu item;

  3. In the Other Settings section, click Security;

  4. In the Security section, click Security Configuration;

  5. In the Advanced Security Configuration section, click Access Controls;

  6. open the Access to functions folder and then, in the Use profiles drop-down list, select No security ;

  7. Confirm this setting.

Manually register Watchdoc ScanCare on a Lexmark device (eSFv3, eSFv4)

  1. Access the web administration interface of the Lexmark device (by entering its IP address in a web browser);

  2. On the device's web administration page, click the Settings menu;

  3. In the Other Settings section, click App :

  4. In the App interface, in the Manage App section, click Manage App. :

  5. at the bottom of the App. interface, click New App...

  6. In the Install New App. section, click Browse to go to the [...]DoxenseScanCareAddIn folder;

  7. in this folder, select the DrivveImage.fls file, then click Open :

  8. once the file is displayed in the selection field, click on the Start button :

  9. When the message indicates that the application is correctly installed, click on the App. tab. :

  10. under the App tab, click ScanCare :

  11. on the Configure tab of the interface, for the Server parameter, enter the IP address of the Watchdoc ScanCare server ;

  12. server, then click Apply.

è Watchdoc ScanCare registration on the Lexmark device is complete.

Manually register Watchdoc ScanCare on a Lexmark device (eSFv5, Android-based eSFv6)

Follow these steps:

  1. Access the Lexmark device's administration web interface (by entering its IP address in a web browser);On the administration web page, click the Applications menu entry:In the Application section, click Installed Apps:Click the Install App button.In the Install App. section, click Browse to navigate to the [...]DoxenseScanCareAddIn folder;Select the DrivveImageAndroid.fls file on the Watchdoc ScanCare serverClick Install. When the installation complete message appears, click OK:In the Application section, click on the ScanCare section to deploy it;

  2. Click on Configure :

  3. Configure :

    • in the Server field: the IP address of the Watchdoc ScanCare server;

    • in the Port field: the port displayed in the Watchdoc ScanCare device management (if the Port is different from the URL of the Watchdoc ScanCare server displayed in the device management);

    • in the Parameters field, enter the usual Watchdoc ScanCare parameters for setting the language of the user interface and keyboard (for example ?ln=fr or ?kb=fr to set the user interface and keyboard in French). Use the ampersand (&) as a separator (e.g. dev=lexmark&ln=en&kb=en).

 

N.B.: for Watchdoc ScanCare 7.x: you may need to enter the dev=lexmark parameter in Parameters for some models. For Watchdoc ScanCare 8.0 and later, this is no longer necessary.

N.B.: Generally, you do not need to enter a value in Parameters. Language parameters should only be entered to force a language if it is not active with the default server and machine settings.

 

  1. Click Apply to confirm.

èWatchdoc ScanCare registration on the Lexmark device is complete.
