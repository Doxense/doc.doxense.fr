Compatibilités

Périphériques d'impression supportés

Watchdoc ScanCare® est compatible avec les périphériques d'impression et de numérisation des constructeurs suivants.

Canon® ScanFront® (scanners)

  • ScanFront® 220/220P

  • ScanFront® 300/ 300P

  • ScanFront® 330 (à partir de la version Watchdoc ScanCare 7.0)

  • ScanFront® 400 (à partir de la version Watchdoc ScanCare 7.0.740).

Fujitsu® (scanners)

  • fi-6010

  • ScanSnap N1800

  • N7100

Sur les N7100, le clavier embarqué génère des erreurs de positionnement. Nous recommandons donc l'utilisation d'un clavier USB pour une utilisation optimale de Watchdoc ScanCare®.

Hewlett Pakard

ScanCare supporte les périphériques Hewlett Packard compatibles avec la couche périphérique OXPd.NET (Open Extensibility Platform) listés dans le guide de compatibilité suivant : https://d1dt3lk9ptjcdc.cloudfront.net/documentation/hp_device_compatibility_guide.pdf.

 

L'écran du périphérique doit supporter une résolution minimale

  • soit de 800 x 600 pixels ;

  • soit de 1024 x 600 pixels.

 

Les séries Hewlett Packard actuellement diffusées qui remplissent ces conditions sont Oriani, Phantom, Pharos, PhotonR and Pulsar. (pour tous ces écrans, les versions avec ou sans clavier coulissant sont supportés).

Les périphériques Hewlett Packard sont compatibles à partir de la version 8.4 de ScanCare. 
Selon certaines informations HP According to recent (2019-03-21) HP information, only 8" panels are currently supporting the required resolutions.

Méthode d'installation

Pour les périphériques Hewlett Packard®, l'installation automatique depuis l'interface d'administration du périphérique est supportée.

Communication réseau

Les ports suivants sont nécessaires pour établir la communication entre Watchdoc ScanCare® et le périphérique :

  • TCP 80, 9000 (http)

  • TCP 443, 4443 (https)

  • TCP 7627 (OXPd)

  • UDP 161 (SNMP)

  • TCP 9100 (Impression directe)

Konica-Minolta®

Les périphériques Konica Minolta® sont supportés depuis la version 7.0.500.

Tous les périphériques Konica Minolta® dotés d'un navigateur web sont supportés.

Les périphériques non dotés d'un navigateur web doivent disposer des composants suivants :

  • Kit Licence navigateur web LK-101v3

  • Mémoire : UX-204

Le navigateur web doit être activé depuis la page d'administration du périphérique.

Kyocera®

Tous les périphériques compatibles HyPas (A3 et A4) sont supportés.

Lexmark®

Les périphériques Lexmark® sont supportés depuis la version Watchdoc® ScanCare 6.1.

 

Tous les multifonctions dotés de Flash, sont supportés :

  • compatibles Framework 3.x :X746de, X748de, X748dte, XS748, C748, CS748, X792de, X792dte, XS795, XS796, XS796, XS798, C792e, C792de, C792dte, C792dhe, CS796, X925de, XS925, C925de, X950de, X952de, X952dte, X954de, X954dhe, XS950, XS955, C950de, X548dte, XS548, 6500e

  • compatibles Framework 4.x : MS610de, MS610dte, M3150, MS810de,M5155, M5163, MS812de, M5170, MS911de, M9155, CS510de, CS510dte, CS511,C2132,MX410 Series,MX51x,XM1145, MX610 Series, MX611 Series, XM3150, MX710 Series, XM5163, MX711 Series, XM5170, MX810 Series, XM7155,X M7155X, XM7163, XM7163X, XM7170, XM7170X, MX910de, MX911dte, MX912dxe, XM9145, XM9155, XM9165, CX410 Series, CX510 Series,XC2132, MX6500e Series ;


Tous les multifonctions dotés de la technologie Androïd sont supportés (multifonctions compatibles avec les Framework 5.x et 6.x) :

  • CX725 series,

  • CX820 series,

  • CX825 series,

  • CX860 series

L'information relative aux périphériques Lexmark supportés est susceptible de changer. Pour déterminer le support des versions eSF actuelles pour certains périphériques, consultez les références fournies par Lexmark.

Oki

Tous les périphériques suivants sont supportés :

  • série MC700

  • série ES94/91xx

  • périphériques utilisant la plateform sXP

  • MC873 / MC853, ES8473 / ES8453 (périphériques A3 couleur sXP2)

  • MB562 / MB492, ES5162 / ES4192 (périphériques A4 mono MFP sXP2 ).

 

Pour mettre à jour le dernier firmware des périphériques listés ci-dessus, utilisez le lien suivant :
http://global.okiprintingsolutions.com/FWInterface.nsf/frmSelectLanguage

Pour les multifonctions OKI Din A3, les documents ne peuvent être numérisés qu'au format paysage.

Ricoh

Les périphériques Ricoh® sont supportés à partir de la version Watchdoc® ScanCare 8.2.

 

Watchdoc® ScanCare 8.2. supporte tous les périphériques compatibles SOP. Si un périphérique n'intègre pas la fonction SOP (Smart Operation Panel), vérifiez s'il existe une mise à jour disponible pour le modèle, de manière à ce que Ricoh Smart SDK (successeur du SDK Type-J), la Web API et le WebBrowserNX soient disponibles pour une intégration parfaite de Watchdoc® ScanCare.

Samsung

Tous les périphériques fondés sur XOA avec une taille d'affichage de 7, 8.9 et 10.1 pouces sont supportés.

Sharp

Tous les périphériques compatibles OSA 2 à 5 sont supportés, excepté les séries Jupiter (AR-M351N/U, AR-M451 N/U) et Dragon (AR-M550, AR-M620 et AR-M700) .

La clé AMX2 doit être activée sur le périphérique OSA.

Toshiba

Tous les modèles compatibles avec e-BRIDGE 3 et versions ultérieures sont supportés.

Le Toshiba Embedded Web Browser (EWB) doit être activé sur le périphérique (code GS-1020 sur l'External Interface Enabler). Cette option n'étant pas disponible partout, il convient de vérifier la présence de cette option avec votre référent Toshiba®.

Xerox

Tous les périphériques compatibles EIP sont supportés, incluant les modèles ColorQube™, VersaLink® et AltaLink®inclus sont supportés.
