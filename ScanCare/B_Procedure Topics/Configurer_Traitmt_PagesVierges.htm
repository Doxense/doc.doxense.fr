Configurer la suppression des pages vierges

La fonction Supprimer les pages vierges permet de détecter les pages vierges et de ne pas en tenir compte lors de la numérisation.

Procédure

Pour configurer le traitement des pages vierges :

  1. lancez le programme de configuration de Watchdoc ScanCare® ;

  2. sélectionnez un profil de numérisation existant ou créez-en un ;

  3. rendez-vous dans la section Traitement ;

  4. cochez la case Supprimer les pages vierges puis cliquez sur le bouton Préférences ;

 

  1. Dans la boîte Préférences,

  • Pourcentage max. de pixels blancs : cochez cette case et indiquez le pourcentages de pixels blancs détectés dans la page pour considérer qu'une page est blanche. Notre recommandation : 99.98%.;

  • Marge : cochez cette case pour ignorer les marges lors de l'évaluation des pixels blancs. Par défautk la marge est d'1/14e de chaque côté pour la largeur et d'1/14e pour la hauteur.

     

  1. Cliquez sur le bouton Tester les paramètres pour vérifier l'effet de la configuration :

  2. sélectionnez un document contenant des pages imprimées et des pages blanches et cliquez sur le bouton Ouvrir ;

  3. dans l'interface de test, les pages sont présentées sous forme de miniatures à gauche et détaillées à droite ;

  4. si vous souhaitez ne pas voir les pages blanches, cochez la case Supprimer les pages ;

  5. cliquez sur le bouton Tester pour lancer le test.

    • si la fonction Marge est activée, la marge s'affiche dans le détail de la page testée ;

    • si la fonction Supprimer les pages est activée, seules les pages imprimées sont affichées ;

    • si la fonction Supprimer les pages est désactivée, chaque page détectée comme une page blanche est signalée par une coche verte ;

       

  1. Une fois les paramètres affinés et testés, validez-les en cliquant sur le bouton OK.

  1. Dans l'interface Traitement de l'image > Supprimer les pages vierges, vous pouvez activer ou désactiver l'affichage de ce traitement sur l'écran du périphérique. De cette manière, l'utilisateur peut modifier le paramétrage établi par défaut :cochez la case Option ;dans la liste, sélectionnez :activé pour activer par défaut ce traitement ;déactivé pour activer par défaut ce traitement.
